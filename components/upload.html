拖拽上传

支持拖拽文件上传
将文件拖到此处,或点击上传
支持多文件上传
<!-- 注意：拖拽功能需要使用 JavaScript 实现 -->
<div class="upload-area" id="dragUploadArea">
    <i class="fa-solid fa-cloud-arrow-up upload-icon"></i>
    <div class="upload-text">将文件拖到此处,或点击上传</div>
    <div class="upload-hint">支持多文件上传</div>
</div>

<script>
// 拖拽事件处理
const area = document.getElementById('dragUploadArea');
area.addEventListener('dragover', (e) => {
    e.preventDefault();
    area.classList.add('upload-area-dragging');
});
area.addEventListener('dragleave', () => {
    area.classList.remove('upload-area-dragging');
});
area.addEventListener('drop', (e) => {
    e.preventDefault();
    area.classList.remove('upload-area-dragging');
    // 处理文件
});
</script>

API

Upload Props

参数 说明 类型 默认值
action 上传的地址 string -
accept 接受上传的文件类型 string -
multiple 是否支持多选文件 boolean false
disabled 是否禁用 boolean false
limit 最大上传数量 number -
fileSize 文件大小限制(MB) number -
listType 列表类型 'text' | 'picture' | 'picture-card' 'text'
drag 是否支持拖拽上传 boolean false
showFileList 是否显示文件列表 boolean true
autoUpload 是否自动上传 boolean true
headers 请求头 object -
data 上传时附带的额外参数 object -
name 文件字段名 string 'file'
withCredentials 是否携带cookie boolean false

CSS 类名

类名 说明
upload-wrapper 上传容器
upload-area 上传区域
upload-area-dragging 拖拽状态
upload-icon 上传图标
upload-text 上传文本
upload-hint 提示文本
upload-btn 上传按钮
upload-list 文件列表
upload-list-item 文件列表项
upload-picture-card 图片卡片
upload-disabled 禁用状态

事件

事件名 说明 回调参数
change 文件状态改变时触发 file: File, fileList: File[]
progress 上传进度时触发 event: ProgressEvent, file: File
success 上传成功时触发 response: any, file: File
error 上传失败时触发 error: Error, file: File
preview 点击预览时触发 file: File
remove 移除文件时触发 file: File
