Badge 徽标

用于展示新消息数量或状态标记

基础用法

不同状态的徽标
Primary Success Warning Danger Info Default 
<span class="badge badge-primary">Primary</span>
<span class="badge badge-success">Success</span>
<span class="badge badge-warning">Warning</span>
<span class="badge badge-danger">Danger</span>
<span class="badge badge-info">Info</span>
<span class="badge badge-default">Default</span>

形状

圆形和方形徽标
圆形 方形 Warning Danger 
<span class="badge badge-primary">圆形</span>
<span class="badge badge-success badge-square">方形</span>

小红点

用于提示新消息
<div class="badge-dot">
    <div class="icon-btn"><i class="fa-solid fa-bell"></i></div>
</div>
<div class="badge-dot badge-dot-success">
    <div class="icon-btn"><i class="fa-solid fa-envelope"></i></div>
</div>

独立徽标

不依赖其他元素的独立徽标
未读 已完成 处理中 已失败 新消息 
<span class="badge badge-primary">未读</span>
<span class="badge badge-success">已完成</span>
<span class="badge badge-warning">处理中</span>
<span class="badge badge-danger">已失败</span>
<span class="badge badge-info">新消息</span>

自定义颜色

通过内联样式自定义颜色
粉色 靛蓝 紫色 绿色 
<span class="badge" style="background: #FFE4E6; color: #881337;">粉色</span>
<span class="badge" style="background: #E0E7FF; color: #3730A3;">靛蓝</span>

<span class="badge" style="background: #DCFCE7; color: #166534;">绿色</span>

API

Badge Props

参数 说明 类型 默认值
type 徽标类型 'primary' | 'success' | 'warning' | 'danger' | 'info' | 'default' 'primary'
shape 徽标形状 'circle' | 'square' 'circle'
text 徽标内容 string | number -
dot 是否显示小红点 boolean false
count 数字徽标的值 number -
maxCount 最大值，超过显示 maxCount+ number 99
showZero 当count为0时是否显示 boolean false

CSS 类名

类名 说明
badge 基础徽标样式
badge-primary 主要徽标（紫色）
badge-success 成功徽标（绿色）
badge-warning 警告徽标（黄色）
badge-danger 危险徽标（红色）
badge-info 信息徽标（蓝色）
badge-default 默认徽标（灰色）
badge-square 方形徽标
badge-dot 小红点容器
badge-dot-success 绿色小圆点
badge-dot-warning 黄色小圆点
badge-dot-info 蓝色小圆点
badge-wrapper 数字徽标容器
badge-count 数字徽标样式
badge-count-lg 大号数字徽标
