ConfigProvider 全局配置

为组件提供全局统一的配置，如语言、主题、尺寸等

基础用法

包裹根组件，为所有子组件提供统一配置

默认配置

<div class="config-provider"
     style="--primary-color: #6750A4; --border-radius: 8px;">
    <button class="btn">默认按钮</button>
    <button class="btn">
        <i class="fa-solid fa-download"></i>
        下载
    </button>
</div>

语言配置

配置组件的显示语言

中文配置

总记录数： 1,234 
<div class="config-provider" data-locale="zh-CN">
    <!-- 组件内容 -->
</div>

<script>
const locales = {
    'zh-CN': {
        confirm: '确定',
        cancel: '取消',
        total: '总记录数'
    },
    'en-US': {
        confirm: 'Confirm',
        cancel: 'Cancel',
        total: 'Total Records'
    }
};

function getText(key) {
    const locale = document.documentElement.dataset.locale || 'zh-CN';
    return locales[locale][key] || key;
}
</script>

方向配置

支持 RTL（从右到左）布局

LTR（从左到右）

RTL（从右到左）

<!-- LTR 布局 -->
<div class="config-provider" dir="ltr">
    <button class="btn">
        <i class="fa-solid fa-arrow-left"></i>
        上一步
    </button>
</div>

<!-- RTL 布局 -->
<div class="config-provider" dir="rtl">
    <button class="btn">
        <i class="fa-solid fa-arrow-right"></i>
        上一步
    </button>
</div>

API

ConfigProvider Props

() => HTMLElement
参数 说明 类型 默认值
theme 主题配置 'light' | 'dark' | object 'light'
primaryColor 主题色 string '#6750A4'
locale 语言包配置 object -
componentSize 组件尺寸 'small' | 'middle' | 'large' 'middle'
direction 文本方向 'ltr' | 'rtl' 'ltr'
prefixCls 组件类名前缀 string 'blinko'
getPopupContainer 弹出框（如 Select、DatePicker）挂载的容器 () => document.body
autoInsertSpaceInButton 按钮中两个汉字之间自动插入空格 boolean true
form 统一设置 Form 组件的属性 object -
renderEmpty 自定义空状态 ReactNode -

CSS 变量

变量名 说明 默认值
--primary-color 主题色 #6750A4
--success-color 成功色 #00B42A
--warning-color 警告色 #FF7D00
--error-color 错误色 #F53F3F
--text-primary 主要文本颜色 #1C1C1E
--text-secondary 次要文本颜色 #4E5969
--border-color 边框颜色 #E5E6EB
--bg-primary 主要背景色 #ffffff
--bg-secondary 次要背景色 #F7F8FA
--border-radius 圆角大小 8px
