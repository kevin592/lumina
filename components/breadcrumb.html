Breadcrumb 面包屑

显示当前页面在系统层级结构中的位置

基础用法

最简单的用法
首页 用户中心 用户设置 
<nav class="breadcrumb">
    <a href="#" class="breadcrumb-item">首页</a>
    <span class="breadcrumb-separator"><i class="fa-solid fa-chevron-right"></i></span>
    <a href="#" class="breadcrumb-item">用户中心</a>
    <span class="breadcrumb-separator"><i class="fa-solid fa-chevron-right"></i></span>
    <span class="breadcrumb-item active">用户设置</span>
</nav>

带图标的面包屑

每个层级可以带图标
首页 文档 组件文档 
<nav class="breadcrumb">
    <a href="#" class="breadcrumb-item">
        <i class="fa-solid fa-home breadcrumb-icon"></i>
        首页
    </a>
    <span class="breadcrumb-separator"><i class="fa-solid fa-chevron-right"></i></span>
    <a href="#" class="breadcrumb-item">
        <i class="fa-solid fa-folder breadcrumb-icon"></i>
        文档
    </a>
    <span class="breadcrumb-separator"><i class="fa-solid fa-chevron-right"></i></span>
    <span class="breadcrumb-item active">
        <i class="fa-solid fa-file breadcrumb-icon"></i>
        组件文档
    </span>
</nav>

自定义分隔符

使用斜线或其他符号作为分隔符
首页 / 用户 / 设置
首页 用户 设置
首页 > 用户 > 设置 
<!-- 斜线分隔符 -->
<nav class="breadcrumb">
    <a href="#" class="breadcrumb-item">首页</a>
    <span class="breadcrumb-separator">/</span>
    <span class="breadcrumb-item active">设置</span>
</nav>

<!-- 箭头分隔符 -->
<nav class="breadcrumb">
    <a href="#" class="breadcrumb-item">首页</a>
    <span class="breadcrumb-separator"><i class="fa-solid fa-angle-right"></i></span>
    <span class="breadcrumb-item active">设置</span>
</nav>

带背景的面包屑

添加背景色的面包屑导航
首页 组件 导航组件 
<nav class="breadcrumb breadcrumb-with-bg">
    <a href="#" class="breadcrumb-item">首页</a>
    <span class="breadcrumb-separator"><i class="fa-solid fa-chevron-right"></i></span>
    <span class="breadcrumb-item active">导航组件</span>
</nav>

尺寸

大、中(默认)、小三种尺寸
首页 大号面包屑
首页 默认面包屑
首页 小号面包屑 
<nav class="breadcrumb breadcrumb-lg">...</nav>
<nav class="breadcrumb">...</nav>
<nav class="breadcrumb breadcrumb-sm">...</nav>

路由器模式

配合路由使用，点击自动跳转
首页 用户管理 用户列表 
<nav class="breadcrumb" id="router-breadcrumb">
    <a href="#home" class="breadcrumb-item" data-route="/home">首页</a>
    <span class="breadcrumb-separator"><i class="fa-solid fa-chevron-right"></i></span>
    <a href="#users" class="breadcrumb-item" data-route="/users">用户管理</a>
    <span class="breadcrumb-separator"><i class="fa-solid fa-chevron-right"></i></span>
    <span class="breadcrumb-item active">用户列表</span>
</nav>

<script>
document.querySelectorAll('#router-breadcrumb .breadcrumb-item').forEach(item => {
    item.addEventListener('click', (e) => {
        if (!item.classList.contains('active')) {
            const route = item.dataset.route;
            console.log('Navigate to:', route);
            // 在这里处理路由跳转
        }
    });
});
</script>

API

Breadcrumb Props

参数 说明 类型 默认值
routes 路由数组 Route[] []
separator 分隔符 string | ReactNode '/'
maxItemCount 最大显示项数 number -
size 尺寸 'sm' | 'md' | 'lg' 'md'
showIcon 是否显示图标 boolean false

Route Props

参数 说明 类型 默认值
path 路由路径 string -
breadcrumbName 面包屑名称 string | ReactNode -
icon 图标 ReactNode -
children 子路由 Route[] -

CSS 类名

类名 说明
breadcrumb 基础面包屑样式
breadcrumb-item 面包屑项
breadcrumb-item.active 当前页面项
breadcrumb-separator 分隔符
breadcrumb-icon 图标样式
breadcrumb-with-bg 带背景样式
breadcrumb-lg 大尺寸
breadcrumb-sm 小尺寸
