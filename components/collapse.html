基础用法

可同时展开多个面板

<script>
function toggleCollapse(header) {
    const content = header.nextElementSibling;
    header.classList.toggle('collapse-header-active');
    content.classList.toggle('collapse-content-active');
}
</script>

什么是 Blinko Design System？
Blinko Design System 是一套完整的 UI 设计系统，包含了丰富的组件和设计规范，帮助开发者快速构建美观、一致的界面。
如何使用这些组件？
只需复制对应的 HTML 代码和 CSS 样式到你的项目中即可使用。所有组件都基于原生 HTML 和 CSS 开发，无需额外的框架依赖。
是否支持自定义主题？
是的，所有组件都支持自定义样式。你可以通过修改 CSS 变量或直接覆盖样式来定制符合你需求的主题。 
<script>
function toggleAccordion(header) {
    const collapse = header.closest('.collapse');
    const allHeaders = collapse.querySelectorAll('.collapse-header');
    const allContents = collapse.querySelectorAll('.collapse-content');

    const isActive = header.classList.contains('collapse-header-active');

    allHeaders.forEach(h => h.classList.remove('collapse-header-active'));
    allContents.forEach(c => c.classList.remove('collapse-content-active'));

    if (!isActive) {
        header.classList.add('collapse-header-active');
        header.nextElementSibling.classList.add('collapse-content-active');
    }
}
</script>

幽灵模式

无边框背景的透明折叠面板
这是什么？
这是幽灵模式的折叠面板，没有边框和背景色，适合在浅色背景上使用。
有什么特点？
幽灵模式更加轻盈简洁，适合需要更多留白空间的场景。

API

Collapse Props

参数 说明 类型 默认值
accordion 是否手风琴模式（每次只展开一个） boolean false
bordered 是否显示边框 boolean true
ghost 是否为幽灵模式（无边框背景） boolean false
defaultActiveKey 默认展开的面板 string[] | number[] []
activeKey 当前展开的面板（受控） string[] | number[] -
onChange 切换面板时的回调 (activeKeys: Array) => void -
expandIcon 自定义展开图标 ReactNode | ({isActive: boolean}) => ReactNode -

CollapseItem Props

参数 说明 类型 默认值
key 唯一标识 string | number -
header 面板标题 string | ReactNode -
extra 右侧额外内容 string | ReactNode -
disabled 是否禁用 boolean false

CSS 类名

类名 说明
collapse 基础折叠面板样式
collapse-item 折叠面板项
collapse-header 折叠面板头部
collapse-header-active 激活状态的头部
collapse-title 折叠面板标题
collapse-extra 折叠面板额外内容区
collapse-icon 折叠面板图标
collapse-content 折叠面板内容容器
collapse-content-active 激活状态的内容
collapse-content-box 折叠面板内容
collapse-ghost 幽灵模式
collapse-borderless 无边框模式
