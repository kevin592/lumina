Radio 单选框

用于在一组选项中单选

基础用法

简单的单选框组
<label class="radio-wrapper">
    <div class="radio">
        <input type="radio" name="option" value="A">
        <div class="radio-box"></div>
    </div>
    <span class="radio-label">选项 A</span>
</label>

禁用状态

禁用的单选框
<label class="radio-wrapper disabled">
    <div class="radio">
        <input type="radio" name="option" value="A" disabled>
        <div class="radio-box"></div>
    </div>
    <span class="radio-label">未选中禁用</span>
</label>

尺寸

小、中(默认)、大三种尺寸
<div class="radio radio-sm"></div>
<div class="radio"></div>
<div class="radio radio-lg"></div>

单选框组

垂直和水平排列的单选框组
垂直排列
水平排列
<div class="radio-group">
    <label class="radio-wrapper">
        <div class="radio">
            <input type="radio" name="city" value="beijing">
            <div class="radio-box"></div>
        </div>
        <span class="radio-label">北京</span>
    </label>
    <!-- 更多选项 -->
</div>

按钮样式

按钮形式的单选框
<div class="radio-button-wrapper">
    <input type="radio" name="btnOption" value="left" checked>
    <label class="radio-button-label">左</label>
</div>
<div class="radio-button-wrapper">
    <input type="radio" name="btnOption" value="center">
    <label class="radio-button-label">中</label>
</div>
<div class="radio-button-wrapper">
    <input type="radio" name="btnOption" value="right">
    <label class="radio-button-label">右</label>
</div>

API

Radio Props

参数 说明 类型 默认值
checked 是否选中 boolean false
disabled 是否禁用 boolean false
value 单选框的值 string -
name 表单项名称 string -

CSS 类名

类名 说明
radio-wrapper 单选框容器
radio 基础单选框样式
radio-box 单选框外观
radio-label 单选框标签
radio-sm 小尺寸 (14px)
radio-lg 大尺寸 (22px)
radio-group 单选框组容器
radio-horizontal 水平排列的单选框组
radio-button-wrapper 按钮样式单选框容器
radio-button-label 按钮样式标签
disabled 禁用状态

事件

事件名 说明 回调参数
change 状态改变时触发 event: Event
click 点击时触发 event: Event
