ResizeBox 伸缩框

可以调整大小的容器组件，支持水平和垂直方向

基础用法

拖动中间的分隔条来调整两侧大小
左侧面板

拖动分隔条可以调整宽度。

右侧面板

这是右侧的内容区域。

<div class="resizebox" style="height: 200px;">
    <div class="resizebox-pane" style="width: 50%;">
        <div style="font-weight: 600;">左侧面板</div>
        <p>拖动分隔条可以调整宽度。</p>
    </div>
    <div class="resizebox-trigger" style="width: 8px;"></div>
    <div class="resizebox-pane" style="flex: 1;">
        <div style="font-weight: 600;">右侧面板</div>
        <p>这是右侧的内容区域。</p>
    </div>
</div>

垂直方向

垂直方向的伸缩框
上面板

拖动分隔条可以调整高度。

下面板

这是下方的内容区域。

<div class="resizebox resizebox-vertical" style="height: 300px;">
    <div class="resizebox-pane" style="height: 50%;">
        <div style="font-weight: 600;">上面板</div>
    </div>
    <div class="resizebox-trigger" style="height: 8px;"></div>
    <div class="resizebox-pane" style="flex: 1;">
        <div style="font-weight: 600;">下面板</div>
    </div>
</div>

固定和弹性

左侧固定宽度，右侧自适应
固定 200px

固定宽度的侧边栏

自适应内容

自适应宽度的主内容区域

<div class="resizebox" style="height: 200px;">
    <div class="resizebox-pane" style="width: 200px; flex-shrink: 0;">
        <div style="font-weight: 600;">固定 200px</div>
    </div>
    <div class="resizebox-trigger" style="width: 8px;"></div>
    <div class="resizebox-pane" style="flex: 1;">
        <div style="font-weight: 600;">自适应内容</div>
    </div>
</div>

尺寸限制

设置面板的最小和最大尺寸
左侧面板 (150px - 300px)

有最小和最大宽度限制

右侧面板 (最小 200px)

有最小宽度限制

<div class="resizebox" style="height: 200px;">
    <div class="resizebox-pane"
         style="width: 30%; min-width: 150px; max-width: 300px;">
        <div style="font-weight: 600;">
            左侧面板
            <span style="font-size: 12px;">(150px - 300px)</span>
        </div>
    </div>
    <div class="resizebox-trigger" style="width: 8px;"></div>
    <div class="resizebox-pane" style="flex: 1; min-width: 200px;">
        <div style="font-weight: 600;">
            右侧面板
            <span style="font-size: 12px;">(最小 200px)</span>
        </div>
    </div>
</div>

API

ResizeBox Props

(index: number) => ReactNode(size: ResizeBoxOnResize) => void() => void() => void
参数 说明 类型 默认值
direction 伸缩方向 'horizontal' | 'vertical' 'horizontal'
split 分隔符 ReactNode -
size 面板尺寸数组 number[] | string[] -
defaultSize 面板初始尺寸 number[] | string[] -
min 面板最小尺寸 number[] | string[] -
max 面板最大尺寸 number[] | string[] -
disabled 是否禁用拖拽 boolean false
handle 自定义拖拽手柄 -
onResize 拖拽时触发的回调 -
onResizeStart 开始拖拽时触发的回调 -
onResizeEnd 结束拖拽时触发的回调 -

CSS 类名

类名 说明
resizebox 基础伸缩框样式
resizebox-vertical 垂直方向
resizebox-pane 面板容器
resizebox-trigger 拖拽分隔条
resizebox-handle 拖拽手柄
resizebox-split 带分隔的容器
resizebox-content 内容区域
