Checkbox 复选框

用于在一组选项中进行多选

基础用法

简单的复选框
<label class="checkbox-wrapper">
    <input type="checkbox">
    <span class="checkbox">
        <span class="checkbox-box"></span>
    </span>
    <span class="checkbox-label">选项 A</span>
</label>

禁用状态

禁用的复选框
<label class="checkbox-wrapper disabled">
    <input type="checkbox" disabled>
    <span class="checkbox"></span>
    <span class="checkbox-label">未选中禁用</span>
</label>

尺寸

小、中(默认)、大三种尺寸
<span class="checkbox checkbox-sm"></span>
<span class="checkbox"></span>
<span class="checkbox checkbox-lg"></span>

复选框组

垂直和水平排列的复选框组
垂直排列
水平排列
<div class="checkbox-group">
    <label class="checkbox-wrapper">
        <input type="checkbox" name="fruit" value="apple">
        <span class="checkbox"></span>
        <span class="checkbox-label">苹果</span>
    </label>
    <!-- 更多选项 -->
</div>

全选

可以实现全选和取消全选功能
<label class="checkbox-wrapper">
    <input type="checkbox" id="checkAll" onclick="toggleAll(this)">
    <span class="checkbox"></span>
    <span class="checkbox-label">全选</span>
</label>

<script>
function toggleAll(source) {
    const checkboxes = document.querySelectorAll('.check-item');
    checkboxes.forEach(cb => cb.checked = source.checked);
}
</script>

API

Checkbox Props

参数 说明 类型 默认值
checked 是否选中 boolean false
disabled 是否禁用 boolean false
indeterminate 是否半选状态 boolean false
value 复选框的值 string -
name 表单项名称 string -

CSS 类名

类名 说明
checkbox-wrapper 复选框容器
checkbox 基础复选框样式
checkbox-label 复选框标签
checkbox-sm 小尺寸 (14px)
checkbox-lg 大尺寸 (22px)
checkbox-group 复选框组容器
checkbox-horizontal 水平排列的复选框组
disabled 禁用状态

事件

事件名 说明 回调参数
change 状态改变时触发 event: Event
click 点击时触发 event: Event
