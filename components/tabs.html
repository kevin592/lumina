Tabs 标签页

用于切换不同内容区域的标签页组件

基础用法

基础的标签页

标签1的内容：这是一个基础的标签页示例，用于展示不同内容区域的切换效果。

标签2的内容：点击不同的标签可以切换显示的内容区域。

标签3的内容：标签页组件可以用于各种场景的内容组织。

<div class="tabs">
    <div class="tabs-header">
        <button class="tabs-tab tabs-tab-active">标签1</button>
        <button class="tabs-tab">标签2</button>
        <button class="tabs-tab">标签3</button>
    </div>
    <div class="tabs-content">
        <div class="tabs-panel tabs-panel-active">内容1</div>
        <div class="tabs-panel">内容2</div>
        <div class="tabs-panel">内容3</div>
    </div>
</div>

API

Tabs Props

参数 说明 类型 默认值
type 标签页类型 'line' | 'card' | 'pill' 'line'
size 标签页尺寸 'sm' | 'md' | 'lg' 'md'
activeKey 当前激活的标签 string | number -
defaultActiveKey 初始激活的标签 string | number -
tabPosition 标签位置 'top' | 'right' | 'bottom' | 'left' 'top'
animated 是否启用动画 boolean true

CSS 类名

类名 说明
tabs 基础标签页样式
tabs-header 标签页头部容器
tabs-tab 标签项
tabs-tab-active 激活的标签项
tabs-content 标签页内容容器
tabs-panel 标签页内容面板
tabs-panel-active 激活的内容面板
tabs-card 卡片式标签页
tabs-pill 胶囊式标签页
tabs-sm 小尺寸标签页
tabs-lg 大尺寸标签页
