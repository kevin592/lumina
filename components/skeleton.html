Skeleton 骨架屏

在内容加载时,展示占位图形

基础用法

文本骨架
<div class="skeleton-paragraph">
    <div class="skeleton skeleton-title"></div>
    <div class="skeleton skeleton-text"></div>
    <div class="skeleton skeleton-text"></div>
    <div class="skeleton skeleton-text"></div>
</div>

组合使用

头像 + 文本
<div class="skeleton-row">
    <div class="skeleton skeleton-avatar"></div>
    <div class="skeleton-col">
        <div class="skeleton skeleton-text-lg" style="width: 40%;"></div>
        <div class="skeleton skeleton-text" style="width: 60%;"></div>
    </div>
</div>

卡片骨架

卡片加载状态
<div class="skeleton-card">
    <div class="skeleton skeleton-image" style="height: 180px; margin-bottom: 16px;"></div>
    <div class="skeleton skeleton-title" style="margin-bottom: 12px;"></div>
    <div class="skeleton skeleton-text"></div>
    <div class="skeleton skeleton-text"></div>
    <div class="skeleton skeleton-text" style="width: 70%;"></div>
    <div class="skeleton-row" style="margin-top: 16px;">
        <div class="skeleton skeleton-button"></div>
        <div class="skeleton skeleton-button"></div>
    </div>
</div>

列表骨架

列表加载状态
<div class="skeleton-list">
    <div class="skeleton-list-item">
        <div class="skeleton skeleton-avatar"></div>
        <div class="skeleton-col">
            <div class="skeleton skeleton-text" style="width: 50%; margin-bottom: 8px;"></div>
            <div class="skeleton skeleton-text" style="width: 80%;"></div>
        </div>
    </div>
    <!-- 更多列表项 -->
</div>

表单骨架

表单加载状态
<div class="skeleton-card">
    <div class="skeleton skeleton-title" style="margin-bottom: 24px;"></div>
    <div style="margin-bottom: 16px;">
        <div class="skeleton skeleton-text" style="width: 30%; margin-bottom: 8px;"></div>
        <div class="skeleton skeleton-input"></div>
    </div>
    <!-- 更多表单项 -->
</div>

头像骨架

不同尺寸的头像
<div class="skeleton skeleton-avatar-sm"></div>
<div class="skeleton skeleton-avatar"></div>
<div class="skeleton skeleton-avatar-lg"></div>

动画效果

默认闪烁效果
<!-- 默认带闪烁动画 -->
<div class="skeleton skeleton-title"></div>

<!-- 可选:脉冲动画 -->
<div class="skeleton skeleton-active"></div>

自定义样式

自定义宽高和圆角
<div class="skeleton" style="height: 40px; width: 200px; border-radius: 8px;"></div>
<div class="skeleton" style="height: 60px; width: 100%; border-radius: 12px;"></div>
<div class="skeleton" style="height: 24px; width: 150px; border-radius: 6px;"></div>

API

Skeleton Props

参数 说明 类型 默认值
loading 是否显示骨架屏 boolean true
active 是否展示动画效果 boolean true
avatar 是否显示头像占位图 boolean | AvatarProps false
paragraph 是否显示段落占位图 boolean | ParagraphProps true
title 是否显示标题占位图 boolean | TitleProps true
round 是否为圆角 boolean false
rows 段落占位图行数 number 3

CSS 类名

类名 说明
skeleton 基础骨架样式
skeleton-text 文本行
skeleton-text-lg 大文本行
skeleton-title 标题行
skeleton-paragraph 段落容器
skeleton-avatar 头像(默认)
skeleton-avatar-lg 大头像
skeleton-avatar-sm 小头像
skeleton-button 按钮
skeleton-input 输入框
skeleton-image 图片
skeleton-card 卡片容器
skeleton-row 行容器
skeleton-col 列容器
skeleton-list 列表容器
skeleton-list-item 列表项
skeleton-active 脉冲动画
