Link 链接

文字超链接，用于页面跳转或操作触发

基础用法

基础文字链接
默认链接 次要链接 
<a href="#" class="link">默认链接</a>
<a href="#" class="link link-default">次要链接</a>

不同状态

链接的多种状态颜色
默认链接 成功链接 警告链接 危险链接 禁用链接 
<a href="#" class="link">默认链接</a>
<a href="#" class="link link-success">成功链接</a>
<a href="#" class="link link-warning">警告链接</a>
<a href="#" class="link link-danger">危险链接</a>
<span class="link link-disabled">禁用链接</span>

下划线

下划线显示方式
始终有下划线 悬停时显示下划线 
<a href="#" class="link link-underline">始终有下划线</a>
<a href="#" class="link link-underline-none">悬停时显示下划线</a>

带图标

带图标的链接
查看详情 下载文档 GitHub 外部链接 
<a href="#" class="link link-with-icon">
    <i class="fa-solid fa-arrow-right"></i>
    查看详情
</a>
<a href="#" class="link link-with-icon">
    下载文档
    <i class="fa-solid fa-download"></i>
</a>

文字大小

不同大小的链接
小号链接 默认链接 中号链接 大号链接 特大号链接 
<a href="#" class="link" style="font-size: 12px;">小号链接</a>
<a href="#" class="link" style="font-size: 14px;">默认链接</a>
<a href="#" class="link" style="font-size: 18px;">大号链接</a>

API

Link Props

参数 说明 类型 默认值
type 链接类型 'primary' | 'default' | 'success' | 'warning' | 'danger' 'primary'
href 跳转链接 string -
disabled 是否禁用 boolean false
underline 下划线显示 'always' | 'hover' | 'none' 'hover'
icon 图标类名 string -
target 打开方式 '_blank' | '_self' | '_parent' | '_top' '_self'

CSS 类名

类名 说明
link 基础链接样式
link-default 次要链接
link-success 成功链接
link-warning 警告链接
link-danger 危险链接
link-disabled 禁用状态
link-with-icon 带图标的链接容器
link-underline 始终显示下划线
link-underline-none 悬停时显示下划线
