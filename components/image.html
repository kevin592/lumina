Image 图片

用于展示图片的组件，支持占位符、加载状态等功能

基础用法

基础图片展示
<div class="image" style="width: 200px; height: 150px;">
    <img src="image.jpg" alt="图片" class="image-img">
</div>

形状

不同形状的图片
<div class="image image-rounded" style="width: 150px; height: 150px;">
    <img src="image.jpg" alt="圆角" class="image-img">
</div>
<div class="image image-circle" style="width: 150px; height: 150px;">
    <img src="image.jpg" alt="圆形" class="image-img">
</div>

占位符

图片加载失败时的占位符
暂无图片
点击上传 
<div class="image image-rounded" style="width: 200px; height: 150px;">
    <div class="image-placeholder">
        <i class="fa-solid fa-image"></i>
        <span>暂无图片</span>
    </div>
</div>

预览

鼠标悬浮显示预览按钮
<div class="image image-preview image-rounded" style="width: 200px; height: 150px;">
    <img src="image.jpg" alt="图片" class="image-img">
    <div class="image-preview-overlay">
        <i class="fa-solid fa-eye image-preview-icon"></i>
    </div>
</div>

加载状态

图片加载中的骨架屏效果
<div class="image image-loading image-rounded" style="width: 200px; height: 150px;"></div>

适应方式

不同的图片填充方式

cover

contain

fill

none

<img src="image.jpg" alt="cover" class="image-img" style="object-fit: cover;">
<img src="image.jpg" alt="contain" class="image-img" style="object-fit: contain;">
<img src="image.jpg" alt="fill" class="image-img" style="object-fit: fill;">
<img src="image.jpg" alt="none" class="image-img" style="object-fit: none;">

API

Image Props

参数 说明 类型 默认值
src 图片地址 string -
alt 替代文本 string -
width 宽度 number | string -
height 高度 number | string -
fit 图片填充方式 'fill' | 'contain' | 'cover' | 'none' | 'scale-down' 'cover'
rounded 是否圆角 boolean false
circle 是否圆形 boolean false
preview 是否可预览 boolean false
placeholder 占位符 string | ReactNode -
fallback 加载失败时的图片 string -
loading 是否显示加载状态 boolean false

CSS 类名

类名 说明
image 基础图片容器
image-img 图片元素
image-rounded 圆角图片
image-circle 圆形图片
image-shadow 带阴影的图片
image-placeholder 占位符容器
image-preview 可预览的图片
image-preview-overlay 预览遮罩层
image-preview-icon 预览图标
image-loading 加载状态
