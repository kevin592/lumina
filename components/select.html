Select 选择器

用于从一组选项中选择一个或多个值

基础用法

基本的选择器
<div class="select-wrapper">
    <select class="select">
        <option value="">请选择</option>
        <option value="apple">苹果</option>
        <option value="banana">香蕉</option>
        <option value="orange">橙子</option>
    </select>
    <i class="fa-solid fa-chevron-down select-icon"></i>
</div>

尺寸

大、中(默认)、小三种尺寸
<select class="select select-lg"></select>
<select class="select"></select>
<select class="select select-sm"></select>

禁用状态

禁用的选择器
<select class="select select-disabled" disabled>
    <option value="">禁用的选择器</option>
    ...
</select>

多选

支持多选的选择器
<select class="select select-multiple" multiple>
    <option value="apple">苹果</option>
    <option value="banana" selected>香蕉</option>
    <option value="orange">橙子</option>
    ...
</select>

选项分组

使用 optgroup 对选项进行分组
<select class="select">
    <option value="">请选择水果</option>
    <optgroup label="热带水果">
        <option value="banana">香蕉</option>
        <option value="mango">芒果</option>
    </optgroup>
    <optgroup label="温带水果">
        <option value="apple">苹果</option>
        <option value="orange">橙子</option>
    </optgroup>
</select>

可搜索

带搜索功能的选择器（需要额外实现）
<!-- 注意：完整的搜索功能需要使用 JavaScript 实现 -->
<div class="select-wrapper">
    <select class="select">
        <option value="">搜索并选择...</option>
        <option value="beijing">北京</option>
        ...
    </select>
    <i class="fa-solid fa-chevron-down select-icon"></i>
</div>

API

Select Props

参数 说明 类型 默认值
value 选中值 string | string[] -
multiple 是否多选 boolean false
disabled 是否禁用 boolean false
size 选择器尺寸 'sm' | 'md' | 'lg' 'md'
placeholder 占位文本 string -
name 表单项名称 string -
required 是否必选 boolean false
autocomplete 自动完成 string 'off'

Option Props

参数 说明 类型 默认值
value 选项值 string -
disabled 是否禁用 boolean false
selected 是否选中 boolean false
label 选项标签 string -

CSS 类名

类名 说明
select-wrapper 选择器容器
select 基础选择器样式
select-icon 下拉箭头图标
select-sm 小尺寸
select-lg 大尺寸
select-multiple 多选样式
select-disabled 禁用状态

事件

事件名 说明 回调参数
change 选项改变时触发 event: Event
focus 获得焦点时触发 event: Event
blur 失去焦点时触发 event: Event
