Trigger 触发器

为任何元素添加弹出层，支持多种触发方式

基础用法

点击按钮显示弹出层
这是一个基础的触发器弹出层 
<div class="trigger" id="trigger1">
    <button class="btn" onclick="toggleTrigger('trigger1')">
        点击显示
    </button>
    <div class="trigger-popup trigger-bottom">
        <div class="trigger-arrow"></div>
        <div class="trigger-content">
            这是一个基础的触发器弹出层
        </div>
    </div>
</div>

<script>
function toggleTrigger(id) {
    const trigger = document.getElementById(id);
    trigger.classList.toggle('active');
}
</script>

触发方式

支持点击、悬停、聚焦等多种触发方式
点击显示的弹出层
悬停显示的弹出层
聚焦显示的弹出层
<!-- 点击触发 -->
<div class="trigger" id="trigger2">
    <button class="btn" onclick="toggleTrigger('trigger2')">
        点击触发
    </button>
    <div class="trigger-popup trigger-bottom">...</div>
</div>

<!-- 悬停触发 -->
<div class="trigger" id="trigger3"
     onmouseenter="showTrigger('trigger3')"
     onmouseleave="hideTrigger('trigger3')">
    <button class="btn btn-outline">悬停触发</button>
    <div class="trigger-popup trigger-bottom">...</div>
</div>

<!-- 聚焦触发 -->
<div class="trigger" id="trigger4">
    <button class="btn btn-outline"
            onfocus="showTrigger('trigger4')"
            onblur="hideTrigger('trigger4')">
        聚焦触发
    </button>
    <div class="trigger-popup trigger-bottom">...</div>
</div>

延迟显示

悬停延迟后显示，鼠标移开立即隐藏
提示信息

这是延迟显示的提示信息，鼠标悬停 300ms 后才会出现。 

<div class="trigger" id="trigger10"
     onmouseenter="delayedShow('trigger10')"
     onmouseleave="hideTrigger('trigger10')">
    <button class="btn btn-outline">
        <i class="fa-solid fa-circle-info"></i>
        悬停查看
    </button>
    <div class="trigger-popup trigger-bottom" style="min-width: 250px;">
        <div class="trigger-arrow"></div>
        <div class="trigger-content">
            <strong>提示信息</strong>
            <p>这是延迟显示的提示信息。</p>
        </div>
    </div>
</div>

<script>
let timeoutId;
function delayedShow(id) {
    timeoutId = setTimeout(() => {
        showTrigger(id);
    }, 300); // 延迟 300ms
}

function hideTrigger(id) {
    clearTimeout(timeoutId);
    const trigger = document.getElementById(id);
    trigger.classList.remove('active');
}
</script>

API

Trigger Props

() => HTMLElement
参数 说明 类型 默认值
action 触发行为 'click' | 'hover' | 'focus' | 'contextMenu' 'hover'
popup 弹出内容 ReactNode | () => ReactNode -
popupVisible 是否显示弹出层（受控） boolean false
defaultPopupVisible 初始是否显示弹出层 boolean false
onVisibleChange 显示/隐藏时的回调 (visible: boolean) => void -
placement 弹出位置 'top' | 'left' | 'right' | 'bottom' | 'topLeft' | 'topRight' | 'bottomLeft' | 'bottomRight' 'bottom'
mouseEnterDelay 鼠标移入后延迟显示的时间 number 100
mouseLeaveDelay 鼠标移出后延迟隐藏的时间 number 100
focusDelay 聚焦后延迟显示的时间 number 0
blurDelay 失焦后延迟隐藏的时间 number 100
mask 是否显示遮罩层 boolean false
maskClosable 点击遮罩是否关闭 boolean true
getPopupContainer 弹出层挂载的容器 () => document.body
showArrow 是否显示箭头 boolean true
alignPoint 是否跟随鼠标位置 boolean false
