Input 输入框

用于接收用户输入的文本信息

基础用法

基本的输入框
<input type="text" class="input" placeholder="请输入内容">

尺寸

大、中(默认)、小三种尺寸
<input type="text" class="input input-lg" placeholder="大尺寸输入框">
<input type="text" class="input" placeholder="默认尺寸输入框">
<input type="text" class="input input-sm" placeholder="小尺寸输入框">

禁用状态

禁用的输入框
<input type="text" class="input input-disabled" placeholder="禁用的输入框" disabled>

带图标

前置和后置图标的输入框
<div class="input-wrapper">
    <span class="input-prefix"><i class="fa-solid fa-search"></i></span>
    <input type="text" class="input" placeholder="搜索...">
</div>

<div class="input-wrapper input-with-icon">
    <input type="text" class="input" placeholder="点击清空" oninput="toggleClear(this)">
    <i class="fa-solid fa-xmark input-icon input-clear" onclick="clearInput(this)"></i>
</div>

带前后缀

带前缀和后缀标签的输入框
http:// .com 
<div class="input-group">
    <span class="input-addon">http://</span>
    <input type="text" class="input" placeholder="请输入网址">
    <span class="input-addon">.com</span>
</div>

密码输入框

带显示/隐藏密码功能
<div class="input-wrapper input-with-icon">
    <input type="password" class="input" id="passwordInput" placeholder="请输入密码">
    <i class="fa-solid fa-eye input-icon" onclick="togglePassword(this)"></i>
</div>

<script>
function togglePassword(icon) {
    const input = icon.previousElementSibling;
    if (input.type === 'password') {
        input.type = 'text';
        icon.classList.replace('fa-eye', 'fa-eye-slash');
    } else {
        input.type = 'password';
        icon.classList.replace('fa-eye-slash', 'fa-eye');
    }
}
</script>

文本域

多行文本输入
<textarea class="input" rows="4" placeholder="请输入多行文本内容"></textarea>

受控输入

实时显示输入值
输入的值: 
<input type="text" class="input" id="controlledInput" oninput="updateValue()">
<div>输入的值: <strong id="inputValue"></strong></div>

<script>
function updateValue() {
    const input = document.getElementById('controlledInput');
    const display = document.getElementById('inputValue');
    display.textContent = input.value;
}
</script>

字数限制

限制输入字符数量
0 / 50 
<textarea class="input" id="limitInput" rows="3" maxlength="50"
    placeholder="最多输入50个字符" oninput="updateCount()"></textarea>
<div style="text-align: right; font-size: 12px; color: #9CA3AF;">
    <span id="charCount">0</span> / 50
</div>

<script>
function updateCount() {
    const input = document.getElementById('limitInput');
    const count = document.getElementById('charCount');
    count.textContent = input.value.length;
}
</script>

API

Input Props

参数 说明 类型 默认值
type 输入框类型 'text' | 'password' | 'email' | 'number' | 'tel' | 'url' 'text'
placeholder 占位文本 string -
disabled 是否禁用 boolean false
readonly 是否只读 boolean false
maxlength 最大字符长度 number -
size 输入框尺寸 'sm' | 'md' | 'lg' 'md'
value 输入框值(受控) string -
defaultValue 输入框默认值(非受控) string -
name 表单项名称 string -
rows 文本域行数(textarea) number -
autocomplete 自动完成 'on' | 'off' 'off'

CSS 类名

类名 说明
input 基础输入框样式
input-sm 小尺寸输入框
input-lg 大尺寸输入框
input-disabled 禁用状态
input-wrapper 输入框容器
input-prefix 前置图标容器
input-suffix 后置图标容器
input-group 输入框组合容器
input-addon 前缀或后缀标签
input-with-icon 带图标的输入框容器
input-icon 输入框内图标
input-clear 清空按钮图标
