触发方式

点击触发（需要JavaScript支持）
点击触发的提示
另一个提示
<div class="tooltip-wrapper" onclick="toggleTooltip(this)">
    <button class="btn">点击显示</button>
    <div class="tooltip tooltip-top" style="opacity: 0; visibility: hidden;">提示</div>
</div>

<script>
function toggleTooltip(wrapper) {
    const tooltip = wrapper.querySelector('.tooltip');
    tooltip.style.opacity = tooltip.style.opacity === '1' ? '0' : '1';
    tooltip.style.visibility = tooltip.style.visibility === 'visible' ? 'hidden' : 'visible';
}
</script>

API

Tooltip Props

参数 说明 类型 默认值
placement 提示位置 'top' | 'left' | 'right' | 'bottom' | 'topLeft' | 'topRight' | 'bottomLeft' | 'bottomRight' 'top'
trigger 触发方式 'hover' | 'click' | 'focus' 'hover'
title 提示内容 string | ReactNode -
color 背景颜色 string '#1D2129'
visible 是否显示（受控模式） boolean -
onVisibleChange 显示/隐藏时的回调 (visible: boolean) => void -
mouseEnterDelay 鼠标移入后延迟显示时间（毫秒） number 100
mouseLeaveDelay 鼠标移出后延迟隐藏时间（毫秒） number 100
