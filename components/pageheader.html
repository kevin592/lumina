PageHeader 页头

页头位于页容器的顶部，让用户快速理解当前页面的信息

基础用法

标准页头

页面标题

这是页面的副标题描述
<div class="page-header">
    <div class="page-header-title-row">
        <div>
            <h1 class="page-header-title">页面标题</h1>
            <div class="page-header-subtitle">这是页面的副标题描述</div>
        </div>
        <div class="page-header-extra">
            <button class="btn btn-primary">主要操作</button>
            <button class="btn btn-outline">次要操作</button>
        </div>
    </div>
</div>

带返回按钮

添加返回链接
返回列表

详情页

<div class="page-header">
    <a href="#" class="page-header-back">
        <i class="fa-solid fa-arrow-left"></i>
        返回列表
    </a>

        <div>
            <h1 class="page-header-title">详情页</h1>
        </div>

            <button class="btn btn-primary">保存</button>
            <button class="btn btn-ghost">取消</button>
        </div>
    </div>
</div>

带面包屑

配合面包屑导航使用
首页 用户管理 用户详情

带内容区域

添加详细的描述内容

项目文档

前端组件库设计规范
这是页面的详细描述内容，可以包含多行文字说明。 用于向用户传达当前页面的用途和重要信息。
文档 前端 UI组件 
<div class="page-header">
    <div class="page-header-title-row">
        <div style="flex: 1;">
            <h1 class="page-header-title">项目文档</h1>
            <div class="page-header-subtitle">前端组件库设计规范</div>
            <div class="page-header-content">
                这是页面的详细描述内容...
            </div>
            <div class="page-header-tags">
                <span class="page-header-tag">文档</span>
                <span class="page-header-tag">前端</span>
                <span class="page-header-tag">UI组件</span>
            </div>
        </div>
        <div class="page-header-extra">
            <button class="btn btn-outline">导出</button>
            <button class="btn btn-primary">发布</button>
        </div>
    </div>
</div>

API

PageHeader Props

参数 说明 类型 默认值
title 标题 string | ReactNode -
subTitle 副标题 string | ReactNode -
backIcon 返回图标 ReactNode -
onBack 返回按钮点击事件 () => void -
extra 操作区 ReactNode | ReactNode[] -
tags 标签列表 ReactNode[] -
footer 页脚 ReactNode -
avatar 头像 ReactNode -
breadcrumb 面包屑配置 BreadcrumbProps -
ghost 是否为幽灵模式 boolean false

CSS 类名

类名 说明
page-header 基础页头样式
page-header-back 返回按钮
page-header-title-row 标题行容器
page-header-title 标题样式
page-header-subtitle 副标题样式
page-header-content 内容区域
page-header-extra 额外操作区
page-header-tags 标签容器
page-header-tag 标签样式
page-header-avatar 头像样式
page-header-with-breadcrumb 带面包屑的页头
