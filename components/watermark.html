图片水印

使用图片作为水印

图片水印示例

可以使用Logo或图标作为水印。

支持图片水印,适用于品牌宣传和版权保护。 
Watermark.init({
    container: 'watermark-3',
    image: '/path/to/logo.png',
    width: 120,
    height: 60,
    gap: [100, 100]
});

自定义样式

自定义颜色、字体、旋转角度等

自定义水印样式

可以自定义水印的各种属性。

包括文字颜色、字体大小、旋转角度、间距等。 
Watermark.init({
    container: 'watermark-4',
    text: 'Blinko Design System',
    fontSize: 20,
    fontWeight: 'bold',
    color: 'rgba(103, 80, 164, 0.15)',
    rotate: -30,
    gap: [120, 100]
});

完全覆盖

整个页面添加水印
// 为整个页面添加水印
Watermark.init({
    container: document.body,
    text: '机密文档',
    fontSize: 24,
    color: 'rgba(0, 0, 0, 0.08)',
    gap: [200, 150]
});

动态更新

动态更新水印内容

动态水印

点击按钮更新水印内容。

// 更新水印
watermarkInstance.update({
    text: '新的水印文本',
    color: 'rgba(255, 59, 48, 0.15)'
});

防篡改

防止通过开发者工具删除水印

防篡改水印

尝试删除水印元素会自动重新创建。

使用 MutationObserver 监听 DOM 变化, 当水印被删除或修改时自动恢复。 
Watermark.init({
    container: 'watermark-6',
    text: '防篡改水印',
    monitor: true,  // 启用监控
    onRemoved: function() {
        console.log('水印被移除,正在恢复...');
    }
});

API

Watermark Props

参数 说明 类型 默认值
text 水印文字,可以是字符串或字符串数组 string | string[] -
image 水印图片 string -
width 水印宽度 number 120
height 水印高度 number 64
rotate 旋转角度 number -22
zIndex 层级 number 1
image 水印图片 URL string -
fontSize 字体大小 number 16
fontWeight 字体粗细 'normal' | 'light' | 'bold' 'normal'
fontStyle 字体样式 'none' | 'normal' | 'italic' | 'oblique' 'normal'
fontFamily 字体 string 'sans-serif'
textAlign 对齐方式 'left' | 'center' | 'right' 'center'
textBaseline 文本基线 'top' | 'middle' | 'bottom' 'middle'
gap 水印间距 [x, y] [number, number] [100, 100]
offset 水印偏移 [x, y] [number, number] [0, 0]
color 水印颜色 string 'rgba(0, 0, 0, 0.15)'
monitor 是否监控水印 DOM 变化 boolean false

update 更新水印 options: Partial<WatermarkOptions>
