API

Table Props

参数 说明 类型 默认值
bordered 是否显示边框 boolean false
size 表格尺寸 'sm' | 'md' | 'lg' 'md'
striped 是否显示斑马纹 boolean false
hoverable 是否显示悬浮效果 boolean false
data 数据源 Array<object> []
columns 列配置 Column[] []

CSS 类名

类名 说明
table 基础表格样式
table-bordered 带边框的表格
table-striped 斑马纹表格
table-hover 悬浮效果表格
table-sm 小尺寸表格
table-lg 大尺寸表格
table-container 表格容器（用于横向滚动）
status-badge 状态徽标基础样式
status-active 激活状态（绿色）
status-inactive 未激活状态（灰色）
status-warning 警告状态（黄色）
status-error 错误状态（红色）
