Popconfirm 气泡确认

点击元素时,弹出气泡确认框,用于确认关键操作

基础用法

默认气泡确认
确定要删除吗?
<div class="popconfirm-wrapper">
    <button class="btn btn-danger" onclick="togglePopconfirm(this)">删除</button>
    <div class="popconfirm">
        <div class="popconfirm-arrow"></div>
        <div class="popconfirm-icon warning">
            <i class="fa-solid fa-triangle-exclamation"></i>
        </div>
        <div class="popconfirm-message">确定要删除吗?</div>
        <div class="popconfirm-buttons">
            <button class="popconfirm-btn popconfirm-btn-cancel">取消</button>
            <button class="popconfirm-btn popconfirm-btn-confirm danger">确定</button>
        </div>
    </div>
</div>

不同类型

警告、信息、错误三种类型
此操作可能影响系统性能,是否继续?
确认要执行此操作吗?
此操作不可恢复,确定要删除吗?
<!-- 警告类型 -->
<div class="popconfirm-icon warning">
    <i class="fa-solid fa-triangle-exclamation"></i>
</div>

<!-- 信息类型 -->
<div class="popconfirm-icon info">
    <i class="fa-solid fa-circle-info"></i>
</div>

<!-- 错误类型 -->
<div class="popconfirm-icon error">
    <i class="fa-solid fa-circle-xmark"></i>
</div>

自定义内容

可以自定义确认框的内容和按钮
自定义标题
这是一段自定义的确认信息,可以包含更多的内容和说明。 
<div class="popconfirm">
    <div class="popconfirm-arrow"></div>
    <div class="popconfirm-message">
        <strong>自定义标题</strong><br>
        这是一段自定义的确认信息,可以包含更多的内容和说明。
    </div>
    <div class="popconfirm-buttons">
        <button class="popconfirm-btn popconfirm-btn-cancel">稍后提醒</button>
        <button class="popconfirm-btn popconfirm-btn-cancel">不再提醒</button>
        <button class="popconfirm-btn popconfirm-btn-confirm">立即处理</button>
    </div>
</div>

自定义按钮文字

自定义确认和取消按钮的文字
确定要退出当前账号吗?
<div class="popconfirm-buttons">
    <button class="popconfirm-btn popconfirm-btn-cancel">继续使用</button>
    <button class="popconfirm-btn popconfirm-btn-confirm danger">退出登录</button>
</div>

异步关闭

点击确定后异步关闭
提交后将进入审核流程,确认提交吗?
function asyncConfirm(btn) {
    // 显示加载状态
    btn.textContent = '提交中...';
    btn.disabled = true;

    // 模拟异步操作
    setTimeout(() => {
        closePopconfirm(btn);
        showMessage('提交成功!');
    }, 1500);
}

API

Popconfirm Props

参数 说明 类型 默认值
visible 是否显示 boolean false
title 确认框标题 string -
description 确认框描述 string -
icon 图标类型 'warning' | 'info' | 'error' 'warning'
okText 确认按钮文字 string '确定'
cancelText 取消按钮文字 string '取消'
onConfirm 点击确认回调 () => void | Promise<void> -
onCancel 点击取消回调 () => void -
asyncConfirm 是否异步关闭 boolean false

CSS 类名

类名 说明
popconfirm-wrapper 包装容器
popconfirm 气泡确认框
popconfirm-arrow 箭头
popconfirm-icon 图标容器
popconfirm-message 消息内容
popconfirm-buttons 按钮组
popconfirm-btn 按钮基础样式
popconfirm-btn-cancel 取消按钮
popconfirm-btn-confirm 确认按钮
