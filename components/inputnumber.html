InputNumber 数字输入

专门用于输入数字的输入框

基础用法

基本的数字输入框
<div class="input-number-wrapper">
    <button class="input-number-btn" onclick="decrease(this)">
        <i class="fa-solid fa-minus"></i>
    </button>
    <input type="number" class="input-number-input" value="1" min="0">
    <button class="input-number-btn" onclick="increase(this)">
        <i class="fa-solid fa-plus"></i>
    </button>
</div>

尺寸

小、中(默认)、大三种尺寸
<div class="input-number-wrapper input-number-sm"></div>
<div class="input-number-wrapper"></div>
<div class="input-number-wrapper input-number-lg"></div>

禁用状态

禁用的数字输入框
<div class="input-number-wrapper input-number-disabled">
    <button class="input-number-btn" disabled></button>
    <input type="number" class="input-number-input" value="1" disabled>
    <button class="input-number-btn" disabled></button>
</div>

步长设置

每次增减的步长值
步长 1
步长 5
步长 0.1
<input type="number" class="input-number-input" value="1" step="1">
<input type="number" class="input-number-input" value="5" step="5">
<input type="number" class="input-number-input" value="1.0" step="0.1">

API

InputNumber Props

参数 说明 类型 默认值
value 当前值 number 0
min 最小值 number -Infinity
max 最大值 number Infinity
step 步长 number 1
precision 数值精度 number -
disabled 是否禁用 boolean false
size 输入框尺寸 'sm' | 'md' | 'lg' 'md'
placeholder 占位文本 string -
controls 是否显示增减按钮 boolean true
stringMode 字符串模式 boolean false
formatter 格式化函数 function -
parser 解析函数 function -

CSS 类名

类名 说明
input-number-wrapper 数字输入容器
input-number-btn 增减按钮
input-number-input 数字输入框
input-number-sm 小尺寸
input-number-lg 大尺寸
input-number-disabled 禁用状态

事件

事件名 说明 回调参数
change 值改变时触发 value: number
focus 获得焦点时触发 event: Event
blur 失去焦点时触发 event: Event
