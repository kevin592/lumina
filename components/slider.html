Slider 滑动条

用于在数值范围内选择一个值

基础用法

基本的滑动条
当前值: 50 
<div class="slider-wrapper">
    <div class="slider">
        <div class="slider-track" style="width: 50%;"></div>
        <div class="slider-handle" style="left: 50%;"></div>
    </div>
    <div class="slider-value">
        当前值: <span class="slider-value-number">50</span>
    </div>
</div>

尺寸

小、中(默认)、大三种尺寸
小尺寸
默认尺寸
大尺寸
<div class="slider slider-sm"></div>
<div class="slider"></div>
<div class="slider slider-lg"></div>

禁用状态

禁用的滑动条
当前值: 50 
<div class="slider-wrapper slider-disabled">
    <div class="slider">
        <div class="slider-track" style="width: 50%;"></div>
        <div class="slider-handle" style="left: 50%;"></div>
    </div>
</div>

范围选择

双滑块范围选择
范围: 30 - 70 
<div class="slider slider-range">
    <div class="slider-track" style="left: 30%; width: 40%;"></div>
    <div class="slider-handle slider-handle-1" style="left: 30%;"></div>
    <div class="slider-handle slider-handle-2" style="left: 70%;"></div>
</div>

API

Slider Props

参数 说明 类型 默认值
min 最小值 number 0
max 最大值 number 100
value 当前值 number | number[] 0
step 步长 number 1
disabled 是否禁用 boolean false
range 是否双滑块模式 boolean false
marks 刻度标记 object -
vertical 是否垂直模式 boolean false
showDots 是否显示步长点 boolean false
tooltip 是否显示提示 boolean true

CSS 类名

类名 说明
slider-wrapper 滑动条容器
slider 基础滑动条样式
slider-track 已滑动部分轨道
slider-handle 滑块手柄
slider-mark 刻度标记
slider-dots 步长点容器
slider-dot 步长点
slider-value 值显示区域
slider-sm 小尺寸
slider-lg 大尺寸
slider-disabled 禁用状态
slider-range 范围选择模式

事件

事件名 说明 回调参数
change 值改变时触发 value: number | number[]
input 拖动时触发 value: number | number[]
