API

Steps Props

参数 说明 类型 默认值
current 当前步骤 number 0
initial 初始步骤 number 0
direction 步骤条方向 'horizontal' | 'vertical' 'horizontal'
size 大小 'default' | 'small' 'default'
status 当前步骤的状态 'wait' | 'process' | 'finish' | 'error' 'process'
progressDot 点状步骤条 boolean | (item: StepItem, index: number) => ReactNode false
onChange 点击步骤时的回调 (current: number) => void -

Step Props

参数 说明 类型 默认值
title 标题 string | ReactNode -
subTitle 副标题 string | ReactNode -
description 步骤的详情描述 string | ReactNode -
icon 步骤图标 ReactNode -
status 指定状态 'wait' | 'process' | 'finish' | 'error' -
disabled 禁用点击 boolean false

CSS 类名

类名 说明
steps 基础步骤条样式
step-item 步骤项
step-item.process 进行中状态
step-item.finish 已完成状态
step-item.error 错误状态
step-icon 步骤图标
step-tail 连接线
step-content 内容区域
step-title 标题
step-description 描述
steps-vertical 垂直方向
steps-small 小尺寸
step-dot 点状步骤条
