Drawer 抽屉

从屏幕一侧滑出的浮动面板,用于展示更多内容或操作

基础用法

从右侧滑出

抽屉标题

这是一个基础的抽屉组件。抽屉可以从屏幕的一侧滑出,用于展示额外的内容或操作。

抽屉适合用于需要保留当前页面上下文的场景,用户可以在不离开当前页面的情况下查看或编辑信息。

与对话框相比,抽屉可以展示更多的内容,且不会完全遮挡页面。

<div class="drawer-overlay" id="drawer-id">
    <div class="drawer drawer-right drawer-md">
        <div class="drawer-header">
            <h3 class="drawer-title">抽屉标题</h3>
            <button class="drawer-close">
                <i class="fa-solid fa-xmark"></i>
            </button>
        </div>
        <div class="drawer-body">
            <p>抽屉内容</p>
        </div>
        <div class="drawer-footer">
            <button class="btn btn-ghost">取消</button>
            <button class="btn btn-primary">确定</button>
        </div>
    </div>
</div>

无页脚

不需要底部操作按钮

无页脚抽屉

这是一个没有页脚的抽屉,适用于简单的信息展示场景。

用户可以通过点击右上角的关闭按钮或点击遮罩层来关闭抽屉。

这种类型的抽屉适合用于查看详情、阅读文档等不需要用户进行确认操作的场景。

<div class="drawer drawer-right drawer-md">
    <div class="drawer-header">
        <h3 class="drawer-title">无页脚抽屉</h3>
        <button class="drawer-close">
            <i class="fa-solid fa-xmark"></i>
        </button>
    </div>
    <div class="drawer-body">
        <p>抽屉内容</p>
    </div>
</div>

自定义页脚

自定义底部操作按钮

自定义页脚

这是一个带有自定义页脚的抽屉。你可以根据需要添加任意数量的按钮。

<div class="drawer-footer">
    <button class="btn btn-ghost">取消</button>
    <button class="btn btn-secondary">上一步</button>
    <button class="btn btn-primary">下一步</button>
</div>

API

Drawer Props

参数 说明 类型 默认值
visible 抽屉是否可见 boolean false
placement 抽屉位置 'top' | 'bottom' | 'left' | 'right' 'right'
size 抽屉尺寸 'sm' | 'md' | 'lg' | 'xl' 'md'
title 抽屉标题 string -
closable 是否显示关闭按钮 boolean true
maskClosable 点击遮罩层是否关闭 boolean true
onClose 关闭时的回调 () => void -

CSS 类名

类名 说明
drawer-overlay 遮罩层
drawer 抽屉容器
drawer-right 从右侧滑出
drawer-left 从左侧滑出
drawer-top 从上方滑出
drawer-bottom 从下方滑出
drawer-sm 小尺寸 (300px)
drawer-md 中尺寸 (500px)
drawer-lg 大尺寸 (700px)
drawer-xl 超大尺寸 (900px)
drawer-header 抽屉头部
drawer-title 抽屉标题
drawer-close 关闭按钮
drawer-body 抽屉内容
drawer-footer 抽屉底部

从上方

这是从上方滑出的抽屉,适合用于展示通知或简短的信息。

从下方

这是从下方滑出的抽屉,适合移动端或需要用户输入的场景。

从左侧

这是从左侧滑出的抽屉。在某些场景下,左侧抽屉可能更符合用户的阅读习惯。

从右侧

这是从右侧滑出的抽屉,是最常见的抽屉形式。
