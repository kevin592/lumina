Notification 通知

在系统右上角浮显示全局通知提醒信息

基础用法

四种基本类型
Notification.success({
    title: '操作成功',
    description: '您的数据已保存'
});

Notification.info({
    title: '系统通知',
    description: '您有新的消息'
});

Notification.warning({
    title: '警告',
    description: '请注意此操作'
});

Notification.error({
    title: '错误',
    description: '操作失败,请重试'
});

带操作按钮

在通知中添加操作按钮
Notification.info({
    title: '系统更新',
    description: '有新版本可用,是否立即更新?',
    actions: [
        { text: '稍后', primary: false },
        { text: '立即更新', primary: true }
    ]
});

自定义持续时间

设置通知显示时长
Notification.info({
    title: '系统通知',
    description: '10秒后关闭',
    duration: 10000
});

// 不自动关闭
Notification.info({
    title: '重要通知',
    description: '需要手动关闭',
    duration: 0
});

带元信息

显示额外的时间信息
Notification.success({
    title: '文件上传成功',
    description: '文件 report.pdf 已上传完成',
    meta: '2024-01-15 14:30'
});

可关闭

默认可关闭
Notification.info({
    title: '系统通知',
    description: '这是一条可关闭的通知',
    closable: true
});

通知堆叠

同时显示多条通知
// 多条通知会自动堆叠显示
Notification.success({ title: '通知1', description: '第一条通知' });
Notification.info({ title: '通知2', description: '第二条通知' });
Notification.warning({ title: '通知3', description: '第三条通知' });

API

Notification 方法

方法名 说明 参数
Notification.success 成功通知 config: NotificationConfig
Notification.info 信息通知 config: NotificationConfig
Notification.warning 警告通知 config: NotificationConfig
Notification.error 错误通知 config: NotificationConfig
Notification.config 全局配置 options: NotificationGlobalConfig
Notification.destroy 销毁所有通知 -

NotificationConfig 参数

参数 说明 类型 默认值
title 通知标题 string -
description 通知内容 string -
duration 持续时间(毫秒) number 4500
closable 是否可关闭 boolean true
actions 操作按钮 Action[] -
meta 元信息 string -
onClose 关闭时的回调 () => void -

CSS 类名

类名 说明
notification-container 通知容器
notification 通知基础样式
notification-success 成功类型
notification-info 信息类型
notification-warning 警告类型
notification-error 错误类型
notification-header 通知头部
notification-icon 图标样式
notification-content 内容区域
notification-title 标题样式
notification-description 描述样式
notification-close 关闭按钮
notification-footer 底部操作区
