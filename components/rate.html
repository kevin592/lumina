Rate 评分

用于评分或评价

基础用法

基本的评分组件
评分: 3 
<div class="rate-wrapper">
    <i class="fa-solid fa-star rate-star active"></i>
    <i class="fa-solid fa-star rate-star active"></i>
    <i class="fa-solid fa-star rate-star active"></i>
    <i class="fa-regular fa-star rate-star"></i>
    <i class="fa-regular fa-star rate-star"></i>
    <span class="rate-value">评分: <span class="rate-value-number">3</span></span>
</div>

尺寸

小、中(默认)、大三种尺寸
<div class="rate-wrapper rate-sm">...</div>
<div class="rate-wrapper">...</div>
<div class="rate-wrapper rate-lg">...</div>

半星

支持半星评分
评分: 3.5 
<!-- 注意：半星需要使用 CSS 或 JavaScript 实现特殊的半星效果 -->
<div class="rate-wrapper">
    <i class="fa-solid fa-star rate-star active"></i>
    <i class="fa-solid fa-star rate-star active"></i>
    <i class="fa-solid fa-star rate-star active"></i>
    <!-- 半星 -->
    <i class="fa-regular fa-star rate-star"></i>
    <i class="fa-regular fa-star rate-star"></i>
    <span class="rate-value">评分: 3.5</span>
</div>

禁用状态

禁用的评分组件
<div class="rate-wrapper rate-disabled">
    <i class="fa-solid fa-star rate-star active"></i>
    <i class="fa-solid fa-star rate-star active"></i>
    <!-- 更多星星 -->
</div>

自定义数量

自定义星星数量
评分: 3
评分: 6
评分: 10 
<!-- 3 星评分 -->
<div class="rate-wrapper">
    <!-- 3 颗星星 -->
</div>

<!-- 7 星评分 -->
<div class="rate-wrapper">
    <!-- 7 颗星星 -->
</div>

<!-- 10 星评分 -->
<div class="rate-wrapper">
    <!-- 10 颗星星 -->
</div>

API

Rate Props

参数 说明 类型 默认值
count 星星总数 number 5
value 当前评分值 number 0
allowHalf 是否允许半星 boolean false
disabled 是否禁用 boolean false
size 星星尺寸 'sm' | 'md' | 'lg' 'md'
character 自定义字符 string | slot 'star'
showValue 是否显示评分值 boolean false
readonly 是否只读 boolean false
clearable 是否可以再次点击清除 boolean false

CSS 类名

类名 说明
rate-wrapper 评分容器
rate-star 星星样式
rate-star.active 激活状态(已选中)
rate-star.half 半星状态
rate-value 评分值显示
rate-value-number 评分值数字
rate-sm 小尺寸
rate-lg 大尺寸
rate-disabled 禁用状态

事件

事件名 说明 回调参数
change 评分改变时触发 value: number
hoverChange 鼠标悬停时触发 value: number
