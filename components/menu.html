API

Menu Props

参数 说明 类型 默认值
mode 菜单类型 'vertical' | 'horizontal' | 'inline' 'vertical'
theme 主题 'light' | 'dark' 'light'
selectedKeys 当前选中的菜单项 key 数组 string[] []
defaultSelectedKeys 初始选中的菜单项 key 数组 string[] []
openKeys 当前展开的SubMenu菜单项 key 数组 string[] []
defaultOpenKeys 初始展开的SubMenu菜单项 key 数组 string[] []
onClick 点击 MenuItem 时的回调 (item: MenuItem) => void -
inlineIndent inline 模式下的菜单缩进宽度 number 24
collapsed 是否收起菜单 boolean false

MenuItem Props

参数 说明 类型 默认值
key 唯一标识 string -
disabled 是否禁用 boolean false
icon 菜单图标 ReactNode -
danger 是否展示为危险操作 boolean false

SubMenu Props

参数 说明 类型 默认值
key 唯一标识 string -
disabled 是否禁用 boolean false
icon 菜单图标 ReactNode -
title 子菜单标题 string | ReactNode -
onTitleClick 点击子菜单标题时的回调 (info: MenuInfo) => void -

CSS 类名

类名 说明
menu 基础菜单样式
menu-item 菜单项
menu-item.active 激活状态
menu-item.disabled 禁用状态
menu-icon 图标样式
menu-item-arrow 箭头样式
menu-submenu 子菜单容器
menu-horizontal 水平菜单
menu-inline 内联菜单
menu-dark 深色主题
menu-group-title 分组标题
