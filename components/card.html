API

Card Props

参数 说明 类型 默认值
size 卡片尺寸 'sm' | 'md' | 'lg' 'md'
bordered 是否显示边框 boolean false
shadow 阴影级别 'none' | 'shadow' | 'shadow-lg' 'none'
hoverable 是否可悬浮 boolean false
cover 封面图片 string -

CSS 类名

类名 说明
card 基础卡片样式
card-sm 小尺寸卡片 (280px)
card-md 中尺寸卡片 (320px)
card-lg 大尺寸卡片 (380px)
card-shadow 基础阴影
card-shadow-lg 强阴影
card-hover 悬浮效果
card-bordered 显示边框
card-header 卡片头部
card-title 卡片标题
card-subtitle 卡片副标题
card-body 卡片内容区
card-footer 卡片底部
card-actions 卡片操作区
card-cover 封面图片
card-text 卡片文本
card-meta 元信息容器
