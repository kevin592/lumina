API

Descriptions Props

参数 说明 类型 默认值
title 标题 string | ReactNode -
extra 额外操作区 string | ReactNode -
bordered 是否显示边框 boolean false
column 每行显示的项目数（可以是数字或对象） number | { xxl: number; xl: number; lg: number; md: number; sm: number; xs: number; } 3
size 尺寸 'sm' | 'md' | 'lg' 'md'
layout 布局方式 'horizontal' | 'vertical' 'horizontal'
colon 标签后是否显示冒号 boolean true
items 描述列表数据 DescriptionsItem[] []

DescriptionsItem Props

参数 说明 类型 默认值
label 标签文本 string | ReactNode -
children 内容 ReactNode -
span 占据的列数 number 1
labelStyle 标签样式 CSSProperties -
contentStyle 内容样式 CSSProperties -

CSS 类名

类名 说明
descriptions 基础描述列表样式
descriptions-header 描述列表头部
descriptions-title 描述列表标题
descriptions-extra 描述列表额外操作区
descriptions-view 描述列表内容视图
descriptions-row 描述列表行
descriptions-item 描述列表项
descriptions-item-label 描述列表项标签
descriptions-item-content 描述列表项内容
descriptions-item-colon 标签后显示冒号
descriptions-bordered 带边框的描述列表
descriptions-sm 小尺寸描述列表
descriptions-lg 大尺寸描述列表
