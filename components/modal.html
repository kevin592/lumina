Modal 对话框

模态对话框,在保留当前页面状态的情况下,展示需要用户处理的信息或任务

基础用法

基本的对话框

对话框标题

这是一个基础的对话框内容。对话框可以包含任何内容,包括文本、表单、图片等。

对话框是模态的,这意味着用户必须与对话框交互后才能继续操作页面。

<div class="modal-overlay" id="basic-modal">
    <div class="modal">
        <div class="modal-header">
            <h3 class="modal-title">对话框标题</h3>
            <button class="modal-close" onclick="closeModal('basic-modal')">
                <i class="fa-solid fa-xmark"></i>
            </button>
        </div>
        <div class="modal-body">
            <p>对话框内容</p>
        </div>
        <div class="modal-footer">
            <button class="btn btn-ghost">取消</button>
            <button class="btn btn-primary">确定</button>
        </div>
    </div>
</div>

不同尺寸

小、默认、中、大、全屏五种尺寸
<div class="modal modal-sm"><!-- 小对话框 --></div>
<div class="modal"><!-- 默认对话框 --></div>
<div class="modal modal-lg"><!-- 中对话框 --></div>
<div class="modal modal-xl"><!-- 大对话框 --></div>
<div class="modal modal-fullscreen"><!-- 全屏对话框 --></div>

确认框

带有确认图标的对话框
<div class="modal-body">
    <i class="fa-solid fa-circle-exclamation modal-confirm-icon info"></i>
    <div>
        <p>确认要执行此操作吗?</p>
    </div>
</div>

自定义页脚

可以自定义底部操作按钮

自定义页脚

这是一个带有自定义页脚的对话框。你可以根据需要添加任意数量的按钮。

<div class="modal-footer">
    <button class="btn btn-ghost">取消</button>
    <button class="btn btn-secondary">上一步</button>
    <button class="btn btn-primary">下一步</button>
</div>

无页脚

不需要底部操作按钮的对话框

无页脚对话框

这是一个没有页脚的对话框,适用于展示信息或通知类场景。

用户可以通过点击右上角的关闭按钮或遮罩层来关闭对话框。

<div class="modal">
    <div class="modal-header">
        <h3 class="modal-title">无页脚对话框</h3>
        <button class="modal-close">
            <i class="fa-solid fa-xmark"></i>
        </button>
    </div>
    <div class="modal-body">
        <p>对话框内容</p>
    </div>
</div>

API

Modal Props

参数 说明 类型 默认值
visible 对话框是否可见 boolean false
title 对话框标题 string -
size 对话框尺寸 'sm' | 'md' | 'lg' | 'xl' | 'fullscreen' 'md'
closable 是否显示关闭按钮 boolean true
maskClosable 点击遮罩层是否关闭 boolean true
onOk 点击确定回调 () => void -
onCancel 点击取消回调 () => void -
afterClose 对话框关闭后的回调 () => void -

CSS 类名

类名 说明
modal-overlay 遮罩层
modal 对话框容器
modal-sm 小尺寸对话框
modal-lg 中大尺寸对话框
modal-xl 大尺寸对话框
modal-fullscreen 全屏对话框
modal-header 对话框头部
modal-title 对话框标题
modal-close 关闭按钮
modal-body 对话框内容
modal-footer 对话框底部
modal-confirm 确认框样式
modal-confirm-icon 确认图标

方法

方法名 说明 参数
openModal 打开对话框 id: string
closeModal 关闭对话框 id: string
confirm 显示确认框 options: ConfirmOptions

小对话框

这是一个小尺寸的对话框,宽度为 400px。

默认对话框

这是默认尺寸的对话框,宽度为 520px。

中对话框

这是中大尺寸的对话框,宽度为 720px。

适合展示更多内容或复杂的表单。

大对话框

这是大尺寸的对话框,宽度为 900px。

适合展示大量内容或需要更大空间的场景。

全屏对话框

这是全屏对话框,占据整个视口。

适合需要完整展示内容的场景,如图表预览、文档编辑等。
