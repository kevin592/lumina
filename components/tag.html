Tag 标签

用于分类和标记的标签组件

基础用法

不同颜色的标签
默认 主要 成功 警告 危险 信息 
<span class="tag tag-default">默认</span>
<span class="tag tag-primary">主要</span>
<span class="tag tag-success">成功</span>
<span class="tag tag-warning">警告</span>
<span class="tag tag-danger">危险</span>
<span class="tag tag-info">信息</span>

圆角标签

使用 rounded 类名创建圆角标签
默认 主要 成功 警告 危险 
<span class="tag tag-default tag-rounded">默认</span>
<span class="tag tag-primary tag-rounded">主要</span>
<span class="tag tag-success tag-rounded">成功</span>

可关闭

带关闭按钮的标签
可关闭标签 成功 警告 危险 
<span class="tag tag-primary tag-closable" onclick="this.style.display='none'">
    可关闭标签
    <span class="tag-close"><i class="fa-solid fa-xmark"></i></span>
</span>

可选中

点击可切换选中状态
标签1 标签2 标签3 标签4 
<span class="tag tag-default tag-checkable" onclick="toggleTag(this)">标签1</span>
<span class="tag tag-default tag-checkable tag-checked" onclick="toggleTag(this)">标签2</span>

尺寸

小、中(默认)、大三种尺寸
小标签 中标签 大标签 
<span class="tag tag-primary tag-sm">小标签</span>
<span class="tag tag-primary">中标签</span>
<span class="tag tag-primary tag-lg">大标签</span>

标签组合

组合使用不同类型的标签
Vue.js React TypeScript Node.js Python 
<span class="tag tag-primary tag-rounded">Vue.js</span>
<span class="tag tag-success tag-rounded">React</span>
<span class="tag tag-info tag-rounded">TypeScript</span>

API

Tag Props

参数 说明 类型 默认值
type 标签类型 'default' | 'primary' | 'success' | 'warning' | 'danger' | 'info' 'default'
rounded 是否为圆角标签 boolean false
closable 是否可关闭 boolean false
checkable 是否可选中 boolean false
checked 是否选中（用于可选中标签） boolean false
size 标签尺寸 'sm' | 'md' | 'lg' 'md'
onClose 关闭时的回调 () => void -
onChange 选中状态改变时的回调 (checked: boolean) => void -

CSS 类名

类名 说明
tag 基础标签样式
tag-default 默认标签（灰色）
tag-primary 主要标签（紫色）
tag-success 成功标签（绿色）
tag-warning 警告标签（黄色）
tag-danger 危险标签（红色）
tag-info 信息标签（蓝色）
tag-rounded 圆角标签
tag-closable 可关闭标签
tag-close 关闭按钮
tag-checkable 可选中标签
tag-checked 已选中状态
tag-sm 小尺寸标签
tag-lg 大尺寸标签
