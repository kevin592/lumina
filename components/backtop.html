<script>
const scrollContainer = document.getElementById('scrollDemo1');
const backTopBtn = document.getElementById('backTop1');

scrollContainer.addEventListener('scroll', () => {
    if (scrollContainer.scrollTop > 100) {
        backTopBtn.classList.add('show');
    } else {
        backTopBtn.classList.remove('show');
    }
});

function scrollToTop(containerId) {
    const container = document.getElementById(containerId);
    container.scrollTo({ top: 0, behavior: 'smooth' });
}
</script>

<script>
const scrollDemo4 = document.getElementById('scrollDemo4');
const backTop4 = document.getElementById('backTop4');

scrollDemo4.addEventListener('scroll', () => {
    // 滚动超过100px后显示按钮
    if (scrollDemo4.scrollTop > 100) {
        backTop4.classList.add('show');
    } else {
        backTop4.classList.remove('show');
    }
});
</script>

API

BackTop Props

() => HTMLElement | HTMLElement
参数 说明 类型 默认值
visibilityHeight 滚动高度达到此参数值才出现 number 400
onClick 点击按钮的回调函数 () => void -
target 设置需要监听其滚动事件的元素 () => window
duration 回到顶部所需时间（ms） number 450
easing 动画缓动函数 string 'linear'

CSS 类名

类名 说明
backtop 基础返回顶部样式
backtop.show 显示状态
backtop-default 默认样式
backtop-primary 主要色样式
backtop-custom 自定义样式
