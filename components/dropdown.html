Dropdown 下拉菜单

向下弹出的菜单列表，用于承载更多操作

基础用法

最简单的下拉菜单
个人中心 设置 消息通知
退出登录 
<div class="dropdown" id="dropdown1">
    <button class="btn btn-primary" onclick="toggleDropdown('dropdown1')">
        点击展开
        <i class="fa-solid fa-chevron-down"></i>
    </button>
    <div class="dropdown-menu">
        <a href="#" class="dropdown-item">个人中心</a>
        <a href="#" class="dropdown-item">设置</a>
        <div class="dropdown-divider"></div>
        <a href="#" class="dropdown-item danger">退出登录</a>
    </div>
</div>

触发方式

支持点击和悬停两种触发方式
选项1 选项2 选项3
选项1 选项2 选项3 
<!-- 点击触发 -->
<div class="dropdown" id="dropdown2">
    <button class="btn btn-outline" onclick="toggleDropdown('dropdown2')">
        点击触发
    </button>
    <div class="dropdown-menu">...</div>
</div>

<!-- 悬停触发 -->
<div class="dropdown" id="dropdown3"
     onmouseenter="showDropdown('dropdown3')"
     onmouseleave="hideDropdown('dropdown3')">
    <button class="btn btn-outline">悬停触发</button>
    <div class="dropdown-menu">...</div>
</div>

API

Dropdown Props

参数 说明 类型 默认值
trigger 触发方式 'click' | 'hover' | 'contextMenu' 'click'
placement 弹出位置 'top' | 'bottom' | 'left' | 'right' 'bottom'
align 对齐方式 'left' | 'right' | 'center' 'left'
disabled 是否禁用 boolean false
visible 是否可见（受控） boolean false
onVisibleChange 显示状态变化回调 (visible: boolean) => void -

Menu Item Props

参数 说明 类型 默认值
key 唯一标识 string -
icon 图标 ReactNode -
disabled 是否禁用 boolean false
danger 是否危险操作 boolean false
onClick 点击事件 () => void -

CSS 类名

类名 说明
dropdown 基础下拉菜单样式
dropdown.active 激活状态
dropdown-menu 菜单容器
dropdown-item 菜单项
dropdown-item.disabled 禁用状态
dropdown-item.danger 危险操作样式
dropdown-divider 分隔线
dropdown-header 分组标题
dropdown-right 右对齐
dropdown-top 向上弹出
dropdown-item-icon 图标样式
dropdown-item-arrow 箭头样式
