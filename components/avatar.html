Avatar 头像

用于展示用户或对象的信息，支持图片、图标或字符

基础用法

使用图片、图标或字符作为头像
U
用户
<div class="avatar avatar-md">
    <img src="avatar.png" alt="用户">
</div>
<div class="avatar avatar-md">
    <i class="fa-solid fa-user"></i>
</div>
<div class="avatar avatar-md">U</div>
<div class="avatar avatar-md">用户</div>

尺寸

四种尺寸：小、中(默认)、大、超大
小
中
大
超大
<div class="avatar avatar-sm">小</div>
<div class="avatar avatar-md">中</div>
<div class="avatar avatar-lg">大</div>
<div class="avatar avatar-xl">超大</div>

形状

圆形、方形、圆角方形
方
<div class="avatar avatar-lg">...</div>
<div class="avatar avatar-lg avatar-square">...</div>
<div class="avatar avatar-lg avatar-rect">...</div>

头像组

头像层叠展示
<div class="avatar-group">
    <div class="avatar avatar-md"><img src="..."></div>
    <div class="avatar avatar-md"><img src="..."></div>
    <div class="avatar avatar-md"><img src="..."></div>
</div>

带徽标

在线状态徽标
<div class="avatar-with-badge">
    <div class="avatar avatar-lg"><img src="..."></div>
    <span class="avatar-badge"></span>
</div>

API

Avatar Props

参数 说明 类型 默认值
size 头像尺寸 'sm' | 'md' | 'lg' | 'xl' 'md'
shape 头像形状 'circle' | 'square' | 'rect' 'circle'
src 图片地址 string -
alt 替代文本 string -
icon 图标类名 string -
text 文本内容 string -
gap 头像组中的间距（像素） number 12

CSS 类名

类名 说明
avatar 基础头像样式
avatar-sm 小尺寸 (32px)
avatar-md 中尺寸 (40px)
avatar-lg 大尺寸 (56px)
avatar-xl 超大尺寸 (80px)
avatar-square 方形头像
avatar-rect 圆角方形头像
avatar-group 头像组容器
avatar-with-badge 带徽标的头像容器
avatar-badge 在线状态徽标（绿色）
avatar-badge-offline 离线状态徽标（灰色）
avatar-badge-busy 忙碌状态徽标（红色）
