Timeline 时间轴

用于展示垂直时间序列信息的组件

基础用法

基础时间轴
创建项目
2024-01-01 10:00
项目初始化完成，开始进行需求分析
设计阶段
2024-01-05 14:30
UI/UX设计完成，进入开发阶段
开发完成
2024-01-15 09:00
功能开发完成，准备测试
<div class="timeline">
    <div class="timeline-item">
        <div class="timeline-tail"></div>
        <div class="timeline-dot"></div>
        <div class="timeline-content">
            <div class="timeline-title">创建项目</div>
            <div class="timeline-time">2024-01-01 10:00</div>
            <div class="timeline-description">项目初始化完成</div>
        </div>
    </div>
</div>

API

Timeline Props

参数 说明 类型 默认值
mode 时间轴模式 'left' | 'alternate' | 'right' 'left'
items 时间轴数据 TimelineItem[] []
reverse 是否倒序 boolean false
pending 是否显示幽灵节点（最后一个节点） boolean false

TimelineItem Props

参数 说明 类型 默认值
color 节点颜色 'primary' | 'success' | 'warning' | 'danger' | 'info' 'primary'
dot 自定义节点 ReactNode -
label 标签（可用于放置时间） ReactNode -
title 标题 ReactNode -
description 描述 ReactNode -

CSS 类名

类名 说明
timeline 基础时间轴样式
timeline-item 时间轴项
timeline-tail 时间轴连线
timeline-dot 时间轴节点
timeline-dot-primary 主要节点（紫色）
timeline-dot-success 成功节点（绿色）
timeline-dot-warning 警告节点（黄色）
timeline-dot-danger 危险节点（红色）
timeline-dot-info 信息节点（蓝色）
timeline-content 时间轴内容
timeline-title 时间轴标题
timeline-time 时间轴时间
timeline-description 时间轴描述
timeline-item-pending 未完成状态
timeline-right 右侧时间轴
timeline-alternate 交替时间轴
