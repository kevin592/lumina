Alert 警告提示

用于页面中展示重要的提示信息

基础用法

四种基本类型
成功
这是一条成功的提示信息
信息
这是一条普通的信息提示
警告
这是一条警告提示信息
错误
这是一条错误提示信息
<div class="alert alert-success">
    <i class="fa-solid fa-circle-check alert-icon"></i>
    <div class="alert-content">
        <div class="alert-title">成功</div>
        <div class="alert-description">这是一条成功的提示信息</div>
    </div>
</div>

可关闭

显示关闭按钮
可关闭提示
点击右侧关闭按钮可以关闭此提示
<div class="alert alert-info">
    <i class="fa-solid fa-circle-info alert-icon"></i>
    <div class="alert-content">
        <div class="alert-title">可关闭提示</div>
        <div class="alert-description">点击右侧关闭按钮可以关闭此提示</div>
    </div>
    <i class="fa-solid fa-xmark alert-close"></i>
</div>

不同尺寸

大、默认、小三种尺寸
默认尺寸
这是默认尺寸的提示信息
小尺寸
这是小尺寸的提示信息
<div class="alert alert-info">
    <i class="fa-solid fa-circle-info alert-icon"></i>
    <div class="alert-content">
        <div class="alert-title">默认尺寸</div>
        <div class="alert-description">这是默认尺寸的提示信息</div>
    </div>
</div>

<div class="alert alert-info alert-sm">
    <i class="fa-solid fa-circle-info alert-icon"></i>
    <div class="alert-content">
        <div class="alert-title">小尺寸</div>
        <div class="alert-description">这是小尺寸的提示信息</div>
    </div>
</div>

无图标

不显示图标
无图标提示
这是不显示图标的提示信息
<div class="alert alert-warning alert-without-icon">
    <div class="alert-content">
        <div class="alert-title">无图标提示</div>
        <div class="alert-description">这是不显示图标的提示信息</div>
    </div>
</div>

Banner 形式

顶部横幅样式
操作成功
您的更改已成功保存
<div class="alert alert-success alert-banner">
    <i class="fa-solid fa-circle-check alert-icon"></i>
    <div class="alert-content">
        <div class="alert-title">操作成功</div>
        <div class="alert-description">您的更改已成功保存</div>
    </div>
    <i class="fa-solid fa-xmark alert-close"></i>
</div>

API

Alert Props

参数 说明 类型 默认值
type 警告类型 'success' | 'info' | 'warning' | 'error' 'info'
closable 是否可关闭 boolean false
showIcon 是否显示图标 boolean true
size 尺寸 'default' | 'small' 'default'
banner 是否为 Banner 形式 boolean false
onClose 关闭时的回调 () => void -

CSS 类名

类名 说明
alert 基础警告样式
alert-success 成功类型
alert-info 信息类型
alert-warning 警告类型
alert-error 错误类型
alert-sm 小尺寸
alert-without-icon 无图标
alert-banner Banner 形式
alert-icon 图标样式
alert-content 内容区域
alert-title 标题样式
alert-description 描述样式
alert-close 关闭按钮
