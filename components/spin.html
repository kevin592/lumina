Spin 加载中

用于页面和区块的加载中状态

基础用法

基本加载图标
加载中... 
<div class="spin">
    <span class="spin-dot"><i></i></span>
    <span class="spin-text">加载中...</span>
</div>

不同颜色

多种语义化颜色
主要
成功
警告
危险
信息 
<div class="spin">
    <span class="spin-dot spin-dot-primary"><i></i></span>
    <span class="spin-text">主要</span>
</div>

<div class="spin">
    <span class="spin-dot spin-dot-success"><i></i></span>
    <span class="spin-text">成功</span>
</div>

不同尺寸

小、默认、大三种尺寸
小尺寸
默认
大尺寸 
<div class="spin spin-sm">
    <span class="spin-dot"><i></i></span>
    <span class="spin-text">小尺寸</span>
</div>

<div class="spin">
    <span class="spin-dot"><i></i></span>
    <span class="spin-text">默认</span>
</div>

<div class="spin spin-lg">
    <span class="spin-dot"><i></i></span>
    <span class="spin-text">大尺寸</span>
</div>

容器加载

加载在容器中
加载中... 
<div class="spin-nested" style="width: 200px; height: 120px;">
    <div class="spin-nested-content spin-blur">
        <!-- 你的内容 -->
    </div>
    <div class="spin-nested-spin">
        <span class="spin-dot"><i></i></span>
        <span class="spin-text">加载中...</span>
    </div>
</div>

延迟加载

延迟显示加载状态
function delayedSpin() {
    // 延迟 500ms 显示加载状态
    setTimeout(() => {
        // 显示 Spin
    }, 500);
}

API

Spin Props

参数 说明 类型 默认值
spinning 是否加载中 boolean true
size 尺寸 'sm' | 'md' | 'lg' 'md'
indicator 加载指示器 VNode | string -
tip 提示文字 string -
delay 延迟显示加载(毫秒) number 0
wrapperClassName 包装器的类名 string -

CSS 类名

类名 说明
spin 加载容器
spin-dot 旋转圆点
spin-dot-primary 主要颜色
spin-dot-success 成功颜色
spin-dot-warning 警告颜色
spin-dot-danger 危险颜色
spin-dot-info 信息颜色
spin-text 加载文字
spin-sm 小尺寸
spin-lg 大尺寸
spin-nested 嵌套容器
spin-nested-content 嵌套内容
spin-nested-spin 嵌套加载层
spin-blur 模糊效果
spin-dot-list 点状列表
spin-dot-list-item 点状列表项
spin-pulse 脉冲效果
