Message 全局提示

全局展示操作反馈信息

基础用法

四种基本类型
// 成功提示
Message.success('操作成功!');

// 信息提示
Message.info('这是一条信息提示');

// 警告提示
Message.warning('请注意此操作!');

// 错误提示
Message.error('操作失败,请重试');

加载中

显示加载状态
const hide = Message.loading('加载中...', 0);

// 手动关闭
// hide();

// 或自动关闭(3秒后)
// Message.loading('加载中...', 3000);

持续时间

自定义提示显示时长
Message.success('5秒后关闭', 5000);
Message.info('10秒后关闭', 10000);

可关闭

带关闭按钮的提示
Message.info({
    content: '这是一条可关闭的提示',
    duration: 0,
    closable: true
});

渲染HTML

支持HTML内容
Message.success({
    content: '<strong>操作成功!</strong><br>您的数据已保存',
    duration: 5000
});

回调函数

提示关闭时的回调
Message.success({
    content: '操作成功!',
    duration: 2000,
    onClose: () => {
        console.log('消息已关闭');
    }
});

API

Message 方法

方法名 说明 参数
Message.success 成功提示 config: string | MessageConfig
Message.info 信息提示 config: string | MessageConfig
Message.warning 警告提示 config: string | MessageConfig
Message.error 错误提示 config: string | MessageConfig
Message.loading 加载提示 config: string | MessageConfig
Message.config 全局配置 options: MessageGlobalConfig
Message.destroy 销毁所有消息 -

MessageConfig 参数

参数 说明 类型 默认值
content 提示内容 string | VNode -
duration 持续时间(毫秒) number 3000
onClose 关闭时的回调 () => void -
closable 是否可关闭 boolean false
icon 自定义图标 string | VNode -

类名 说明

message-warning 警告类型
message-error 错误类型
