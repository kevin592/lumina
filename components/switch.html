Switch 开关

用于在两个互斥的状态之间切换

基础用法

基本的开关
<label class="switch-wrapper">
    <div class="switch">
        <input type="checkbox">
        <div class="switch-slider"></div>
    </div>
    <span class="switch-label">关闭</span>
</label>

尺寸

小、中(默认)、大三种尺寸
<div class="switch switch-sm"></div>
<div class="switch"></div>
<div class="switch switch-lg"></div>

禁用状态

禁用的开关
<label class="switch-wrapper disabled">
    <div class="switch">
        <input type="checkbox" disabled>
        <div class="switch-slider"></div>
    </div>
    <span class="switch-label">禁用关闭</span>
</label>

加载状态

加载中的开关
<label class="switch-wrapper switch-loading">
    <div class="switch">
        <input type="checkbox" disabled>
        <div class="switch-slider"></div>
    </div>
    <span class="switch-label">加载中</span>
</label>

文字和图标

带文字说明的开关
<label class="switch-wrapper">
    <div class="switch">
        <input type="checkbox" checked>
        <div class="switch-slider"></div>
    </div>
    <span class="switch-label">推送通知</span>
</label>

API

Switch Props

参数 说明 类型 默认值
checked 是否选中 boolean false
disabled 是否禁用 boolean false
loading 是否加载中 boolean false
size 开关尺寸 'sm' | 'md' | 'lg' 'md'
checkedText 选中时的文字 string -
uncheckedText 未选中时的文字 string -

CSS 类名

类名 说明
switch-wrapper 开关容器
switch 基础开关样式
switch-slider 开关滑块
switch-label 开关标签
switch-sm 小尺寸 (40x24px)
switch-lg 大尺寸 (56x32px)
switch-loading 加载状态
disabled 禁用状态

事件

事件名 说明 回调参数
change 状态改变时触发 checked: boolean
click 点击时触发 event: Event
