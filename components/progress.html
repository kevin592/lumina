Progress 进度条

展示操作的当前进度

基础用法

不同百分比的进度条
30%
50%
70%
100% 
<div class="progress">
    <div class="progress-outer">
        <div class="progress-bar progress-bar-primary" style="width: 30%;"></div>
    </div>
    <span class="progress-text">30%</span>
</div>

不同颜色

多种语义化颜色
主要
成功
警告
危险
信息 
<div class="progress-bar progress-bar-primary"></div>
<div class="progress-bar progress-bar-success"></div>
<div class="progress-bar progress-bar-warning"></div>
<div class="progress-bar progress-bar-danger"></div>
<div class="progress-bar progress-bar-info"></div>

不同尺寸

小、默认、大三种尺寸
小尺寸
默认
大尺寸 
<div class="progress progress-sm"><!-- 小尺寸 --></div>
<div class="progress"><!-- 默认 --></div>
<div class="progress progress-lg"><!-- 大尺寸 --></div>

条纹动画

带动画的条纹进度条
条纹
动画 
<!-- 条纹 -->
<div class="progress-bar progress-bar-primary progress-bar-striped"></div>

<!-- 条纹动画 -->
<div class="progress-bar progress-bar-success progress-bar-striped animated"></div>

带标签

显示进度标签和百分比
文件上传 75%
处理中... 45% 
<div>
    <div class="progress-info">
        <span class="progress-label">文件上传</span>
        <span class="progress-percent">75%</span>
    </div>
    <div class="progress">
        <div class="progress-outer">
            <div class="progress-bar progress-bar-primary" style="width: 75%;"></div>
        </div>
    </div>
</div>

环形进度条

圆形进度展示
30%
50%
70%
100% 
<div class="progress-circle">
    <svg width="100" height="100">
        <circle class="progress-circle-bg" cx="50" cy="50" r="40" stroke-width="8"></circle>
        <circle class="progress-circle-bar progress-bar-primary" cx="50" cy="50" r="40" stroke-width="8"
                stroke-dasharray="251.2" stroke-dashoffset="175.84"></circle>
    </svg>
    <span class="progress-circle-text">30%</span>
</div>

<!-- 计算: stroke-dashoffset = 251.2 * (1 - percentage / 100) -->
<!-- 30%: 251.2 * 0.7 = 175.84
     50%: 251.2 * 0.5 = 125.6
     70%: 251.2 * 0.3 = 75.36
     100%: 251.2 * 0 = 0 -->

API

Progress Props

参数 说明 类型 默认值
percent 进度百分比 number 0
type 进度条类型 'line' | 'circle' 'line'
status 状态 'primary' | 'success' | 'warning' | 'danger' | 'info' 'primary'
size 尺寸 'sm' | 'md' | 'lg' 'md'
striped 是否条纹 boolean false
animated 是否动画 boolean false
showInfo 是否显示百分比 boolean true
label 标签文本 string -

CSS 类名

类名 说明
progress 进度条容器
progress-outer 进度条外层
progress-bar 进度条
progress-bar-primary 主要颜色
progress-bar-success 成功颜色
progress-bar-warning 警告颜色
progress-bar-danger 危险颜色
progress-bar-info 信息颜色
progress-bar-striped 条纹样式
progress-text 百分比文字
progress-info 信息区域
progress-label 标签文字
progress-percent 百分比样式
progress-circle 环形进度条容器
progress-circle-bg 环形背景
progress-circle-bar 环形进度条
progress-circle-text 环形中心文字
