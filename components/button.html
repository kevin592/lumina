Button 按钮

用于触发一个操作或提交表单数据

基础用法

按钮类型
<button class="btn btn-primary">主要按钮</button>
<button class="btn btn-secondary">次要按钮</button>
<button class="btn btn-outline">描边按钮</button>
<button class="btn btn-ghost">幽灵按钮</button>
<button class="btn btn-danger">危险按钮</button>
<button class="btn btn-success">成功按钮</button>

按钮尺寸

大、中(默认)、小三种尺寸
<button class="btn btn-primary btn-lg">大按钮</button>
<button class="btn btn-primary">默认按钮</button>
<button class="btn btn-primary btn-sm">小按钮</button>

图标按钮

带图标的按钮
<button class="btn btn-primary">
    <i class="fa-solid fa-plus"></i>
    新建
</button>
<button class="btn btn-primary btn-round">
    <i class="fa-solid fa-heart"></i>
</button>

加载状态

添加 loading 类名表示加载中
<button class="btn btn-primary btn-disabled">
    <i class="fa-solid fa-spinner fa-spin"></i>
    加载中...
</button>

禁用状态

按钮不可用状态
<button class="btn btn-primary btn-disabled">禁用按钮</button>
<button class="btn btn-secondary btn-disabled">禁用按钮</button>
<button class="btn btn-outline btn-disabled">禁用按钮</button>

API

Button Props

参数 说明 类型 默认值
type 按钮类型 'primary' | 'secondary' | 'outline' | 'ghost' | 'danger' | 'success' 'primary'
size 按钮尺寸 'sm' | 'md' | 'lg' 'md'
disabled 是否禁用 boolean false
loading 是否加载中 boolean false
round 是否为圆形按钮 boolean false
icon 图标类名 string -

CSS 类名

类名 说明
btn 基础按钮样式
btn-primary 主要按钮
btn-secondary 次要按钮
btn-outline 描边按钮
btn-ghost 幽灵按钮
btn-danger 危险按钮
btn-success 成功按钮
btn-sm 小尺寸
btn-lg 大尺寸
btn-disabled 禁用状态
btn-round 圆形按钮
