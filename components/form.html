Form 表单

用于数据收集、验证和提交

基础用法

基本的表单
我们将向您的邮箱发送验证链接
<form class="form">
    <div class="form-item">
        <label class="form-label form-label-required">用户名</label>
        <input type="text" class="form-control" placeholder="请输入用户名">
    </div>
    <div class="form-item">
        <label class="form-label form-label-required">密码</label>
        <input type="password" class="form-control" placeholder="请输入密码">
    </div>
    <div class="form-actions">
        <button type="submit" class="btn btn-primary">提交</button>
        <button type="reset" class="btn btn-secondary">重置</button>
    </div>
</form>

表单验证

带错误提示的表单项
用户名长度至少为 3 个字符
<div class="form-item form-item-has-error">
    <label class="form-label form-label-required">用户名</label>
    <input type="text" class="form-control" value="ab">
    <div class="form-error">用户名长度至少为 3 个字符</div>
</div>

对齐方式

左对齐、右对齐、顶部对齐
<div class="form-item">
    <label class="form-label">标签左对齐</label>
    <input type="text" class="form-control" placeholder="请输入内容">
</div>

复杂表单

包含多种表单元素的复杂表单
最多 200 个字符
<form class="form">
    <div class="form-item">
        <label class="form-label form-label-required">姓名</label>
        <input type="text" class="form-control" placeholder="请输入姓名">
    </div>
    <!-- 更多表单项 -->
    <div class="form-actions">
        <button type="submit" class="btn btn-primary">提交</button>
        <button type="button" class="btn btn-secondary">取消</button>
    </div>
</form>

API

Form Props

参数 说明 类型 默认值
layout 表单布局 'horizontal' | 'vertical' | 'inline' 'vertical'
labelAlign 标签对齐方式 'left' | 'right' | 'top' 'left'
labelWidth 标签宽度 string | number -
size 表单尺寸 'sm' | 'md' | 'lg' 'md'

Form.Item Props

参数 说明 类型 默认值
required 是否必填 boolean false
label 标签文本 string -
extra 额外的提示信息 string -
help 帮助文本 string -
validateStatus 验证状态 'success' | 'warning' | 'error' | 'validating' -
hasFeedback 是否显示反馈图标 boolean false

CSS 类名

类名 说明
form 基础表单样式
form-item 表单项容器
form-label 表单标签
form-label-required 必填标签
form-control 表单控件
form-extra 额外提示信息
form-error 错误提示
form-item-has-error 错误状态
form-actions 操作按钮区域

事件

事件名 说明 回调参数
submit 提交表单时触发 event: Event, values: object
finish 表单验证通过后触发 values: object
finishFailed 表单验证失败后触发 errorInfo: object
reset 重置表单时触发 event: Event
valuesChange 字段值变化时触发 changedValues: object, allValues: object
