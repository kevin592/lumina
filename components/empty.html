Empty 空状态

用于展示空状态和引导操作的占位组件

基础用法

基础空状态
暂无数据
<div class="empty">
    <div class="empty-image">...SVG 图标...</div>
    <div class="empty-description">暂无数据</div>
</div>

带操作

带操作按钮的空状态
还没有任何数据，开始创建吧
<div class="empty">
    <div class="empty-image">...SVG 图标...</div>
    <div class="empty-description">还没有任何数据</div>
    <div class="empty-footer">
        <button class="btn btn-primary">创建数据</button>
    </div>
</div>

不同场景

不同场景的空状态
暂无消息
未找到相关内容
文件夹为空
<div class="empty">
    <div class="empty-image">
        <i class="fa-solid fa-inbox" style="font-size: 64px;"></i>
    </div>
    <div class="empty-description">暂无消息</div>
</div>

尺寸

小、中(默认)、大三种尺寸
小尺寸空状态
中尺寸空状态
大尺寸空状态
<div class="empty empty-simple">...</div>
<div class="empty empty-sm">...</div>
<div class="empty">...</div>
<div class="empty empty-lg">...</div>

自定义图片

使用自定义图片作为空状态
自定义图片的空状态
<div class="empty">
    <div class="empty-image" style="width: 200px; height: 150px;">
        <img src="empty.svg" alt="空状态" style="width: 100%; height: 100%; object-fit: contain;">
    </div>
    <div class="empty-description">自定义图片的空状态</div>
</div>

API

Empty Props

参数 说明 类型 默认值
image 空状态图片（URL或ReactNode） string | ReactNode -
imageStyle 图片样式 CSSProperties -
description 空状态描述文字 string | ReactNode '暂无数据'
children 底部操作区域 ReactNode -

CSS 类名

类名 说明
empty 基础空状态样式
empty-image 空状态图片容器
empty-description 空状态描述文字
empty-footer 空状态底部操作区
empty-sm 小尺寸空状态
empty-lg 大尺寸空状态
empty-simple 简洁空状态
