API

List Props

参数 说明 类型 默认值
bordered 是否显示边框 boolean false
size 列表尺寸 'sm' | 'md' | 'lg' 'md'
split 是否显示分割线 boolean true
hoverable 是否显示悬浮效果 boolean true

CSS 类名

类名 说明
list 基础列表样式
list-bordered 带边框的列表
list-sm 小尺寸列表
list-lg 大尺寸列表
list-item 列表项
list-item-avatar 列表项头像
list-item-content 列表项内容
list-item-title 列表项标题
list-item-desc 列表项描述
list-item-action 列表项操作区
list-item-meta 列表项元信息
list-header 列表头部
list-footer 列表底部
