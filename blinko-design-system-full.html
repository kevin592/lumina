Grid 栅格

栅格可以有效的保证页面的一致性、逻辑性、加强团队协作和统一。

基本用法

栅格系统基于 8 原子单位，提供 12 等分和 24 等分两种系统。

24 栅格系统（原型使用）

  <div class="col-span-1">1</div>
  <div class="col-span-2">2</div>
  <div class="col-span-3">3</div>
  ...
</div>

/* Tailwind 配置 */

ConfigProvider 全局配置

为包裹的组件提供全局配置。

基本用法

ConfigProvider 提供全局的组件前缀、语言等配置。

<ConfigProvider
  prefixCls="blinko"
  componentSize="default"
  locale="zh-CN"
>
  <App />
</ConfigProvider>

/* 可配置项 */
interface ConfigProviderProps {
  prefixCls?: string;          // 组件类名前缀
  componentSize?: string;     // 组件大小
  locale?: string;            // 语言
  autoInsertSpaceInButton?: boolean;  // 自动插入空格
  focusing?: boolean;         // 是否自动聚焦
  empty?: {                  // 空状态配置
    description?: ReactNode;
    img?: string;
    simple?: boolean;
  };
}
